# <a name="pidtagfreebusypublishstart-canonical-property"></a>PidTagFreeBusyPublishStart の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
公開の範囲の開始時刻が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_FREEBUSY_PUBLISH_START  <br/> |
|識別子:  <br/> |0x6847  <br/> |
|データを入力します。  <br/> |PT_LONG  <br/> |
|領域:  <br/> |空き/予約済み  <br/> |
   
## <a name="remarks"></a>備考

このプロパティの値は、1601 年 1 月 1 日世界協定時刻 (UTC) で、午前 0 時以降の分の数です。
