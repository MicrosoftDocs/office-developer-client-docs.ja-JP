<span data-ttu-id="e0457-118">メッセージ ストアプロバイダーとアドレス帳プロバイダーは、テーブル プロパティを取得するための両方の手法をサポートする必要があります。</span><span class="sxs-lookup"><span data-stu-id="e0457-118">Message store and address book providers must support both techniques for retrieving table properties.</span></span> <span data-ttu-id="e0457-119">クライアントが選択肢を持つ必要がある場合に、プロバイダーがこれらのテーブルにアクセスするための 1 つの方法のみをサポートすることはできません。</span><span class="sxs-lookup"><span data-stu-id="e0457-119">It is unacceptable for providers to support only one way for accessing these tables because clients expect to have the choice.</span></span>
