# <a name="pidtagbirthday-canonical-property"></a>PidTagBirthday 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の誕生日が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_BIRTHDAY  <br/> |
|識別子:  <br/> |0x3A42  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>備考

このプロパティは、id と連絡先に関する情報にアクセスを提供するプロパティの 1 つです。 これらのプロパティは、ユーザーおよびユーザーの組織によって定義されます。
