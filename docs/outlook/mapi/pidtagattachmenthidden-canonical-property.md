# <a name="pidtagattachmenthidden-canonical-property"></a>PidTagAttachmentHidden 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
添付ファイルがエンドユーザーに対して非表示になっているかどうかを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_ATTACHMENT_HIDDEN  <br/> |
|識別子:  <br/> |0x7ffe  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |メッセージの添付ファイル  <br/> |
