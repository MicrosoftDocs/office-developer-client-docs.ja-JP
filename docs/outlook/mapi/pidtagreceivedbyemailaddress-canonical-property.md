# <a name="pidtagreceivedbyemailaddress-canonical-property"></a>PidTagReceivedByEmailAddress 標準プロパティ

  
  
**適用対象**: Outlook 
  
メッセージを受信したメッセージのユーザーの電子メール アドレスが含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_RECEIVED_BY_EMAIL_ADDRESS、PR_RECEIVED_BY_EMAIL_ADDRESS_A、PR_RECEIVED_BY_EMAIL_ADDRESS_W  <br/> |
|識別子:  <br/> |0x0076  <br/> |
|データの種類 :   <br/> |PT_UNICODE、PT_STRING8  <br/> |
|領域:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティでは、メッセージを受信したメッセージのユーザーのアドレスのプロパティの例を示します。 着信トランスポート プロバイダーが設定されなければなりません。
