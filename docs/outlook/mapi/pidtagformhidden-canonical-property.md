# <a name="pidtagformhidden-canonical-property"></a>PidTagFormHidden の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
フォームを表示から非表示にする場合は TRUE が含まれています、メニューとダイアログ ボックスを作成します。 
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_FORM_HIDDEN  <br/> |
|識別子:  <br/> |0x3307  <br/> |
|データを入力します。  <br/> |PT_BOOLEAN  <br/> |
|領域:  <br/> |一般的な MAPI  <br/> |
