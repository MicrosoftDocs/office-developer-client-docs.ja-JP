### <a name="identifying-properties-in-mapi"></a><span data-ttu-id="ef8a7-113">MAPI のプロパティの識別</span><span class="sxs-lookup"><span data-stu-id="ef8a7-113">Identifying Properties in MAPI</span></span>

<span data-ttu-id="ef8a7-114">MAPI のすべてのプロパティは、署名されていない 16 ビット値であるプロパティ識別子によって識別されます。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-114">All properties in MAPI are identified by a property identifier that is an unsigned 16-bit value.</span></span> <span data-ttu-id="ef8a7-115">プロパティ識別子とプロパティの種類 (もう 1 つの符号なし 16 ビット値) は、プロパティのプロパティ タグを構成します。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-115">The property identifier and the property type (another unsigned 16-bit value) constitute a property tag for the property.</span></span> 
  
<span data-ttu-id="ef8a7-116">MAPI では、プロパティ タグを使用してプロパティを一意に定義します。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-116">MAPI uses a property tag to uniquely define properties.</span></span> <span data-ttu-id="ef8a7-117">**PR_BUSINESS2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)) や **PR_OFFICE2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber)](pidtagbusiness2telephonenumber-canonical-property.md)など、同じプロパティ タグを持つプロパティは、MAPI では同じプロパティと見なされます。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-117">Properties that have the same property tag, like **PR_BUSINESS2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)) and **PR_OFFICE2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)), are considered identical properties in MAPI.</span></span> <span data-ttu-id="ef8a7-118">MAPI が独自のプロパティに対して定義したプロパティ タグの一覧については、「MAPI ヘッダー ファイル Mapitags.h」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-118">For a list of property tags that MAPI has defined for its own properties, see the MAPI header file, Mapitags.h.</span></span>
  
<span data-ttu-id="ef8a7-119">MAPI はプロパティ識別子を範囲に分割します。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-119">Note that MAPI divides property identifiers into ranges.</span></span> <span data-ttu-id="ef8a7-120">識別子が範囲内にある場合は、その使用と所有権を示します。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-120">Where an identifier falls in the range indicates its use and ownership.</span></span> <span data-ttu-id="ef8a7-121">タグ付きプロパティのプロパティ識別子は、タグ付きプロパティの0x0001から0x7FFF。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-121">The property identifiers of tagged properties fall in the range of 0x0001 to 0x7FFF.</span></span> <span data-ttu-id="ef8a7-122">この範囲内には、MAPI 定義プロパティのプロパティ識別子が含まれるので、このプロパティの範囲は0x0001から0x3FFF。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-122">Within this range are the property identifiers of MAPI-defined properties, which fall in the range of 0x0001 to 0x3FFF.</span></span> <span data-ttu-id="ef8a7-123">名前付きプロパティのプロパティ識別子は、指定されたプロパティの範囲0x8000から0x8FFF。</span><span class="sxs-lookup"><span data-stu-id="ef8a7-123">The property identifiers of named properties fall in the range from 0x8000 to 0x8FFF.</span></span>
