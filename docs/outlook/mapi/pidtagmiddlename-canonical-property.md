# <a name="pidtagmiddlename-canonical-property"></a>PidTagMiddleName 標準プロパティ

  
  
**適用対象**: Outlook

|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_MIDDLE_NAME、PR_MIDDLE_NAME_A、PR_MIDDLE_NAME_W  <br/> |

|データの種類 :   <br/> |PT_UNICODE、PT_STRING8  <br/> |
