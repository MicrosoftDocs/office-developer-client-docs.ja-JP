# <a name="pidnamexsharingcapabilities-canonical-property"></a>PidNameXSharingCapabilities 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
アイテムの共有機能を示す値を格納します。
  
|||
|:-----|:-----|
|フレンドリ名:  <br/> |なし  <br/> |
|プロパティセット:  <br/> |PS_INTERNET_HEADERS  <br/> |
|プロパティ名:  <br/> |X 共有機能  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |共有  <br/> |
   
## <a name="remarks"></a>解説

このプロパティには、メッセージオブジェクトの共有機能を示す値が含まれています。
