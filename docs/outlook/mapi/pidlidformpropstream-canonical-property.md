# <a name="pidlidformpropstream-canonical-property"></a>PidLidFormPropStream 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージと一緒に保存されるカスタム プロパティのデータを表します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidFormPropStream  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x0000851B  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |実行時の構成  <br/> |
