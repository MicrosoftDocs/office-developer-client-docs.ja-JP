# <a name="pidtagincompletecopy-canonical-property"></a>PidTagIncompleteCopy 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
このメッセージが他のメッセージの不完全なコピーである場合は、TRUE が含まれます。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_INCOMPLETE_COPY  <br/> |
|識別子:  <br/> |0x0035  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |一般的なメッセージのプロパティ  <br/> |
