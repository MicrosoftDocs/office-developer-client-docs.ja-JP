# <a name="pidtagrulename-canonical-property"></a>PidTagRuleName 標準プロパティ

  
  
**適用対象**: Outlook 
  
ルールのユーザーが判読できる名前を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_RULE_NAME  <br/> |
|識別子:  <br/> |0x6682  <br/> |
|データの種類 :   <br/> |PT_STRING8  <br/> |
|領域:  <br/> |サーバー側のルール  <br/> |
