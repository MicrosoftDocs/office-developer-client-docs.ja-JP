# <a name="pidtagrecipientdisplayname-canonical-property"></a>PidTagRecipientDisplayName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
**PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)) プロパティの値を格納します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_RECIPIENT_DISPLAY_NAME、PR_RECIPIENT_DISPLAY_NAME_A、PR_RECIPIENT_DISPLAY_NAME_W  <br/> |
|識別子:  <br/> |0x5ff6  <br/> |
|データの種類 :   <br/> |PT_STRING8、PT_UNICODE  <br/> |
|エリア:  <br/> |トランスポート受信者  <br/> |
