# <a name="pidtagtransportproviders-canonical-property"></a>PidTagTransportProviders 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_TRANSPORT_PROVIDERS  <br/> |
|識別子:  <br/> |0x3d02  <br/> |
