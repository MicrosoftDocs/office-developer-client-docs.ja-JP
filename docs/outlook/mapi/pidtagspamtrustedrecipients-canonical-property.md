# <a name="pidtagspamtrustedrecipients-canonical-property"></a>PidTagSpamTrustedRecipients 標準プロパティ
 
**適用対象**: Outlook 2013 | Outlook 2016 
  
信頼できる受信者を表す、セミコロンで区切られた電子メールアドレスとドメインのリストが含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_SPAM_TRUSTED_RECIPIENTS_W  <br/> |
|ロング ID (LID):  <br/> |0x0419  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |スパム  <br/> |
