# <a name="pidlidemail3addresstype-canonical-property"></a>PidLidEmail3AddressType 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
3番目の電子メールアドレスのアドレスの種類を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidEmail3AddrType  <br/> |
|プロパティセット:  <br/> |PSETID_Address  <br/> |
|ロング ID (LID):  <br/> |0x000080a2  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |連絡先  <br/> |
   
## <a name="remarks"></a>解説

このプロパティの値が存在する場合は、有効なアドレスの種類である必要があります。
