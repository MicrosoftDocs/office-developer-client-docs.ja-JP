# <a name="pidlidemail3addresstype-canonical-property"></a>PidLidEmail3AddressType 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
3 番目の電子メール アドレスのアドレスの種類を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidEmail3AddrType  <br/> |
|プロパティを設定します。  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x000080A2  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|領域:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注釈

存在する場合、このプロパティの値は有効なアドレスの種類である必要があります。
