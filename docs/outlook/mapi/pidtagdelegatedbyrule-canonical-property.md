# <a name="pidtagdelegatedbyrule-canonical-property"></a>PidTagDelegatedByRule 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
フォルダーのメッセージがルールによって委任されているかどうかを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_DELEGATED_BY_RULE  <br/> |
|識別子:  <br/> |0x3fe3  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |MAPI の状態  <br/> |
