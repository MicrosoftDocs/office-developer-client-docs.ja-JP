# <a name="pidlidhomeaddresscountrycode-canonical-property"></a>PidLidHomeAddressCountryCode 標準プロパティ

  
  
**適用対象**: Outlook 
  
連絡先の自宅の住所の国コードを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidHomeAddressCountryCode  <br/> |
|プロパティを設定します。  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x000080DA  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|領域:  <br/> |Contact  <br/> |
