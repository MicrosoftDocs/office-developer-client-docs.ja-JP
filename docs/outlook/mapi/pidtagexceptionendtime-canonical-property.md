# <a name="pidtagexceptionendtime-canonical-property"></a>PidTagExceptionEndTime の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
例外が作成されると、最後の日付と時刻、コンピューターのローカル タイム ゾーンの例外を示します。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_EXCEPTION_ENDTIME  <br/> |
|識別子:  <br/> |0x7FFC  <br/> |
|データを入力します。  <br/> |PT_SYSTIME  <br/> |
|領域:  <br/> |クラスによって定義されたメッセージの転送を可能な  <br/> |
