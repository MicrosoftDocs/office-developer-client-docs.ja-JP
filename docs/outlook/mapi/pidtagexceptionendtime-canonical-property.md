# <a name="pidtagexceptionendtime-canonical-property"></a>PidTagExceptionEndTime 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
例外が作成されたときに、コンピューターのローカルタイムゾーンの例外の終了日時を示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_EXCEPTION_ENDTIME  <br/> |
|識別子:  <br/> |0x7ffc  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |メッセージクラスで定義された、非表示テーブル  <br/> |
   
## <a name="remarks"></a>解説

> [!NOTE]
> このプロパティは情報であり、重要な情報に依存することはできません。
