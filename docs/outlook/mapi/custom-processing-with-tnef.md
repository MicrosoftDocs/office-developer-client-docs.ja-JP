# <a name="custom-processing-with-tnef"></a>TNEF を処理するカスタム

**適用されます**: Outlook 2013 |Outlook 2016 
  
トランスポート プロバイダーでは、添付ファイル自体のプロパティを処理、添付ファイルを個別に送信またはメッセージング システムの添付ファイルのモデルからの送信に、カスタム処理を使用できます。 TNEF は、メッセージと添付ファイルを送信し、受信側で再接続するトランスポート プロバイダーを有効にするメカニズムを使用します。
