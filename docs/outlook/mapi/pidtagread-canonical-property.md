# <a name="pidtagread-canonical-property"></a>PidTagRead 標準プロパティ

  
  
**適用対象**: Outlook 
  
テーブルの行が読み取られたかどうかを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_READ  <br/> |
|識別子:  <br/> |0x0E69  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|領域:  <br/> |MAPI 以外から送信できます。  <br/> |
