# <a name="pidtagbusinesshomepage-canonical-property"></a>PidTagBusinessHomePage 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ビジネスのホームページの URL が保存されています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_BUSINESS_HOME_PAGE、PR_BUSINESS_HOME_PAGE_A、PR_BUSINESS_HOME_PAGE_W  <br/> |
|識別子:  <br/> |0x3a51  <br/> |
|データの種類 :   <br/> |PT_UNICODE、PT_STRING8  <br/> |
|エリア:  <br/> |連絡先  <br/> |
   
## <a name="remarks"></a>解説

これらのプロパティは、連絡先のビジネスホームページへのリンクとして使用されます。
