# <a name="pidnameauthor-canonical-property"></a>PidNameAuthor 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
ドキュメントに関連付けられているファイルの作成者を表します。
  
|||
|:-----|:-----|
|フレンドリ名:  <br/> |なし  <br/> |
|プロパティを設定します。  <br/> |PS_PUBLIC_STRINGS  <br/> |
|プロパティ名:  <br/> |Author  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|領域:  <br/> |Common  <br/> |
