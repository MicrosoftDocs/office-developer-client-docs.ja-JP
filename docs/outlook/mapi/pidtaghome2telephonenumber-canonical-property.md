# <a name="pidtaghome2telephonenumber-canonical-property"></a>PidTagHome2TelephoneNumber 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
受信者の自宅のセカンダリ電話番号が含まれています。 
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_HOME2_TELEPHONE_NUMBER、PR_HOME2_TELEPHONE_NUMBER_A、PR_HOME2_TELEPHONE_NUMBER_W  <br/> |
|識別子:  <br/> |0x3a2f  <br/> |
|データの種類 :   <br/> |PT_STRING8、PT_UNICODE  <br/> |
|エリア:  <br/> |MAPI メールユーザー  <br/> |
   
## <a name="remarks"></a>解説

これらのプロパティは、受信者の id とアクセス情報を提供します。 受信者と組織で定義されています。
