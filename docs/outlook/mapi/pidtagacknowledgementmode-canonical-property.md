# <a name="pidtagacknowledgementmode-canonical-property"></a>PidTagAcknowledgementMode の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
メッセージの受信確認モードの識別子が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_ACKNOWLEDGEMENT_MODE  <br/> |
|識別子:  <br/> |0x0001  <br/> |
|データを入力します。  <br/> |PT_LONG  <br/> |
|領域:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>備考

このプロパティは、次の値の 1 つだけ持つことができます。
  
|**値**|**説明**|
|:-----|:-----|
|0  <br/> |手動受信を確認します。  <br/> |
|1  <br/> |自動受信を確認します。  <br/> |
