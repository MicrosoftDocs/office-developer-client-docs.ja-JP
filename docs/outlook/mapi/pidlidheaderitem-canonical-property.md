# <a name="pidlidheaderitem-canonical-property"></a>PidLidHeaderItem 標準プロパティ

  
  
**適用対象**: Outlook 
  
かどうか Outlook のダウンロードがクライアント コンピューターに、メッセージまたはヘッダーとメッセージの本文のヘッダーのみ Exchange キャッシュ モードで Outlook が構成されている場合を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidHeaderItem  <br/> |
