# <a name="pidlidheaderitem-canonical-property"></a>PidLidHeaderItem 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
outlook が Exchange キャッシュモードで構成されている場合に、メッセージのヘッダーのみをクライアントコンピューターにダウンロードするか、メッセージのヘッダーと本文の両方を Microsoft outlook でダウンロードするかを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidheaderitem  <br/> |
|プロパティセット:  <br/> |PSETID_Common  <br/> |
|ロング ID (LID):  <br/> |0x00008578  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |一般的なメッセージング  <br/> |
