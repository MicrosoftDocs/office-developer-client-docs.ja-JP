# <a name="pidlidyomilastname-canonical-property"></a>PidLidYomiLastName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の姓のふりがなを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidYomiLastName  <br/> |
|プロパティセット:  <br/> |PSETID_Address  <br/> |
|ロング ID (LID):  <br/> |0x0000802d  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |連絡先  <br/> |
