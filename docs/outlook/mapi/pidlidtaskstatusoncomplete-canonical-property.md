# <a name="pidlidtaskstatusoncomplete-canonical-property"></a>PidLidTaskStatusOnComplete 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
タスク実施者が、割り当てられたタスクを完了したときに、電子メールメッセージの更新を送信することを要求したかどうかを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidtasksoc  <br/> |
|プロパティセット:  <br/> |PSETID_Task  <br/> |
|ロング ID (LID):  <br/> |0x00008119  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |タスク  <br/> |
