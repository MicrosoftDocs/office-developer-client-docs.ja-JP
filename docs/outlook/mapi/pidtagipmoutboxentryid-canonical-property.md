# <a name="pidtagipmoutboxentryid-canonical-property"></a>PidTagIpmOutboxEntryId の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
標準的な個人間メッセージ (IPM) の [送信トレイ] フォルダーのエントリ id が含まれています。 
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_IPM_OUTBOX_ENTRYID  <br/> |
|識別子:  <br/> |0x35E2  <br/> |
|データを入力します。  <br/> |PT_BINARY  <br/> |
|領域:  <br/> |Folder  <br/> |
   
## <a name="remarks"></a>備考

通常、送信メッセージは送信トレイ フォルダーに作成されます。 個人間メッセージを送信するためには、このフォルダーに配置します。
