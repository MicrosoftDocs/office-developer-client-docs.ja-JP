# <a name="pidtagformcategorysub-canonical-property"></a>PidTagFormCategorySub 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
クライアント アプリケーションによって定義されているフォームのサブカテゴリが含まれています。 
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_FORM_CATEGORY_SUB、PR_FORM_CATEGORY_SUB_A、PR_FORM_CATEGORY_SUB_W  <br/> |
|識別子:  <br/> |0x3305  <br/> |
|データの種類 :   <br/> |PT_STRING8、PT_UNICODE  <br/> |
|領域:  <br/> |一般的な MAPI  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティは、 **PR_FORM_CATEGORY** ([PidTagFormCategory](pidtagformcategory-canonical-property.md)) のプロパティで提供されているメイン フォームのカテゴリに従属します。
