# <a name="pidlidworkaddresscity-canonical-property"></a>PidLidWorkAddressCity 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の勤務先住所の市区町村または地域名を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidWorkAddressCity  <br/> |
|プロパティを設定します。  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x00008046  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
