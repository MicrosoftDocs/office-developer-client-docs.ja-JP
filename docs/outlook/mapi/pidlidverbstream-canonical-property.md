# <a name="pidlidverbstream-canonical-property"></a>PidLidVerbStream 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージへの応答でどのようなユーザーの応答を返信するを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidVerbStream  <br/> |
|プロパティを設定します。  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x00008520  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |実行時の構成  <br/> |
