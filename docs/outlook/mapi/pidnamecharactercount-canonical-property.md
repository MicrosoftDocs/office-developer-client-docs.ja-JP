# <a name="pidnamecharactercount-canonical-property"></a>PidNameCharacterCount 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
文書に添付するファイルの文字数を指定します。
  
|||
|:-----|:-----|
|フレンドリ名:  <br/> |なし  <br/> |
|プロパティセット:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|プロパティ名:  <br/> |CharCount  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |共通  <br/> |
