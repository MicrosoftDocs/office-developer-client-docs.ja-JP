# <a name="pidlidappointmentlastsequence-canonical-property"></a>PidLidAppointmentLastSequence 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
任意の出席者に最後に送信されたシーケンス番号を開催者に示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidapptlastsequence  <br/> |
|プロパティセット:  <br/> |PSETID_Appointment  <br/> |
|ロング ID (LID):  <br/> |0x00008203  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>解説

このプロパティは、出席者に対しては意味がありません。
