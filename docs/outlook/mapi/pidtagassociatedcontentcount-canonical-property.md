# <a name="pidtagassociatedcontentcount-canonical-property"></a>PidTagAssociatedContentCount の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
フォルダーの内容が関連付けられているテーブル内の項目の数が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_ASSOC_CONTENT_COUNT  <br/> |
|識別子:  <br/> |0x3617  <br/> |
|データを入力します。  <br/> |PT_LONG  <br/> |
|領域:  <br/> |Folder  <br/> |
   
## <a name="remarks"></a>備考

このプロパティには、フォルダーの内容のテーブル内のエントリの数が含まれています。
