# <a name="pidlidtaskupdates-canonical-property"></a>PidLidTaskUpdates の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
タスク実施者が割り当てられたタスクが変更されたときにタスクの更新を送信する要求されたかどうかを示します。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |dispidTaskUpdates  <br/> |
|プロパティを設定します。  <br/> |PSETID_Task  <br/> |
|長い ID (LID):  <br/> |0x0000811B  <br/> |
|データを入力します。  <br/> |PT_BOOLEAN  <br/> |
|領域:  <br/> |タスク  <br/> |
