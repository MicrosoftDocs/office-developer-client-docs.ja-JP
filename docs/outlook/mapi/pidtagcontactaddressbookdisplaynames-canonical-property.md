# <a name="pidtagcontactaddressbookdisplaynames-canonical-property"></a>PidTagContactAddressBookDisplayNames の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
メッセージングの連絡先アドレス帳のユーザーの表示名の一覧が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_CONTAB_DISPLAY_NAMES、PR_CONTAB_DISPLAY_NAMES_A、PR_CONTAB_DISPLAY_NAMES_W  <br/> |
|識別子:  <br/> |0x6624  <br/> |
|データを入力します。  <br/> |PT_MV_UNICODE、PT_MV_STRING8  <br/> |
|領域:  <br/> |連絡先のアドレス帳  <br/> |
