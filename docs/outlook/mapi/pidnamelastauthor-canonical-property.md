# <a name="pidnamelastauthor-canonical-property"></a>PidNameLastAuthor 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
添付ファイルの最新の作成者を指定します。
  
|||
|:-----|:-----|
|フレンドリ名:  <br/> |なし  <br/> |
|プロパティセット:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|プロパティ名:  <br/> |LastAuthor  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |共通  <br/> |
