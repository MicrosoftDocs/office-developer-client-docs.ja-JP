# <a name="pidtagreplyrequested-canonical-property"></a>PidTagReplyRequested の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
メッセージの送信者が受信者からの返信を要求した場合、TRUE が格納されます。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_REPLY_REQUESTED  <br/> |
|識別子:  <br/> |0x0C17  <br/> |
|データを入力します。  <br/> |PT_BOOLEAN  <br/> |
|領域:  <br/> |MAPI 受信者  <br/> |
