**適用対象**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidClassDesc  <br/> |
|プロパティを設定します。  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x000085B7  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |メッセージ全般  <br/> |
