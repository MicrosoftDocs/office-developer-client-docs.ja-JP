# <a name="pidtaginternetmailoverrideformat-canonical-property"></a>PidTagInternetMailOverrideFormat 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
送信メールに設定する必要がある添付ファイルを含めることを HTML エンコード方式を示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_INETMAIL_OVERRIDE_FORMAT  <br/> |
|識別子:  <br/> |0x5902  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|領域:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティを分割するサブ部分については、 [[MS OXOMSG]](http://msdn.microsoft.com/library/daa9120f-f325-4afb-a738-28f91049ab3c%28Office.15%29.aspx)を参照してください。
