# <a name="pidlidlogdocumentsaved-canonical-property"></a>PidLidLogDocumentSaved 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ジャーナル処理中にドキュメントを保存するかどうかを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidLogDocSaved  <br/> |
|プロパティを設定します。  <br/> |PSETID_Log  <br/> |
|長い ID (LID):  <br/> |0x0000870F  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |ジャーナル  <br/> |
