# <a name="pidtagabsearchpath-canonical-property"></a>PidTagAbSearchPath の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
名前を解決するのには検索対象のアドレス帳コンテナーのエントリの識別子の一覧が含まれています。 
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_AB_SEARCH_PATH  <br/> |
|識別子:  <br/> |0x3D05  <br/> |
|データを入力します。  <br/> |PT_MV_BINARY  <br/> |
|領域:  <br/> |アドレス帳  <br/> |
   
## <a name="remarks"></a>備考
