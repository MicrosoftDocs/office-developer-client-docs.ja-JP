# <a name="mapisvcinf-message-service-sections"></a>MapiSvc.inf メッセージ サービス セクション

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
mapisvc.inf には、 **[サービス]** セクションに記載されているエントリのそれぞれのメッセージ サービスの 1 つのセクションが含まれています。 これらのセクション内のエントリの 2 種類があります: 1 つ構成されているメッセージ サービスに関連するセクションの名前を一覧表示するため、その他の特定のプロパティを設定するためです。
