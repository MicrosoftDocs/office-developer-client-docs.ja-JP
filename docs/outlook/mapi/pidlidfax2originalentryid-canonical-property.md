# <a name="pidlidfax2originalentryid-canonical-property"></a>PidLidFax2OriginalEntryId 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
連絡先の自宅の fax アドレスの元のエントリ Id を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidFax2OriginalEntryID  <br/> |
|プロパティを設定します。  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x000080C5  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|領域:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは、存在する場合がこの fax アドレスに対応する 1 回限りのエントリ Id を指定しなければなりません。
