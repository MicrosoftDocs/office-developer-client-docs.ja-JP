# <a name="pidtagpstrememberpassword-canonical-property"></a>PidTagPstRememberPassword 標準プロパティ

  
  
**適用対象**: Outlook 
  
プロファイルにパスワードを保存するかどうかを決定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_PST_REMEMBER_PW  <br/> |
|識別子:  <br/> |0x6701  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|領域:  <br/> |パーソナル ストレージ表 (.pst) 内部  <br/> |
