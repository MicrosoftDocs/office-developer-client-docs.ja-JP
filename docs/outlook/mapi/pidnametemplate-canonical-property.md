# <a name="pidnametemplate-canonical-property"></a>PidNameTemplate 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ドキュメントに添付されているファイルのテンプレートを指定します。
  
|||
|:-----|:-----|
|分名:  <br/> |なし  <br/> |
|プロパティ セット:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|プロパティ名:  <br/> |テンプレート  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |共通  <br/> |
