# <a name="pidtagpagertelephonenumber-canonical-property"></a>PidTagPagerTelephoneNumber 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
受信者のポケットベルの電話番号が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_PAGER_TELEPHONE_NUMBER、PR_PAGER_TELEPHONE_NUMBER_A、PR_PAGER_TELEPHONE_NUMBER_W、PR_BEEPER_TELEPHONE_NUMBER、PR_BEEPER_TELEPHONE_NUMBER_A、PR_BEEPER_TELEPHONE_NUMBER_W  <br/> |
|識別子:  <br/> |0x3A21  <br/> |
|データの種類 :   <br/> |PT_UNICODE、PT_STRING8  <br/> |
|領域:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティでは、識別を提供し、受信者の情報にアクセスします。 受信者とその構造によって定義されます。
