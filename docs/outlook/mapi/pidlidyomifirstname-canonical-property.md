# <a name="pidlidyomifirstname-canonical-property"></a>PidLidYomiFirstName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の指定された名前の発音を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidYomiFirstName  <br/> |
|プロパティ セット:  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x0000802C  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
