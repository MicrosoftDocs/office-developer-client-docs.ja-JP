# <a name="pidtagreceivedbyentryid-canonical-property"></a>PidTagReceivedByEntryId 標準プロパティ

  
  
**適用対象**: Outlook 
  
実際にメッセージを受け取るユーザーはメッセージのエントリ id が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_RECEIVED_BY_ENTRYID  <br/> |
|識別子:  <br/> |0x003F  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|領域:  <br/> |アドレス帳  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは、メッセージを受信したメッセージのユーザーのアドレスのプロパティのいずれかです。 受信トランスポート プロバイダーによって設定する必要があります。
  
このプロパティは、X.400 配信エンベロープ メッセージごとの MH_T_ 属性に対応します。
