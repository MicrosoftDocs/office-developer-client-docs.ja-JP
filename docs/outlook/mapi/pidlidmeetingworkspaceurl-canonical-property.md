# <a name="pidlidmeetingworkspaceurl-canonical-property"></a>PidLidMeetingWorkspaceUrl 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
予定表オブジェクトに関連付けられている会議ワークスペースの URL を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidmwsurl  <br/> |
|プロパティセット:  <br/> |PSETID_Appointment  <br/> |
|ロング ID (LID):  <br/> |0x00008209  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>解説

このプロパティは必須ではありません。
