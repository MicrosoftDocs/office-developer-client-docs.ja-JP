# <a name="pidtagjunkphishingenablelinks-canonical-property"></a>PidTagJunkPhishingEnableLinks の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
示す、TRUE の場合、メッセージのフィッシング詐欺のタイムスタンプを無視することです。
