# <a name="pidlidremotetransfertime-canonical-property"></a>PidLidRemoteTransferTime 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
リモート アイテムの秒数で、転送時間を示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidRemoteXferTime  <br/> |
|プロパティを設定します。  <br/> |PSETID_Remote  <br/> |
|長い ID (LID):  <br/> |0x00008F04  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|領域:  <br/> |リモート ・ メッセージ  <br/> |
