# <a name="pidlidcontactuserfield2-canonical-property"></a>PidLidContactUserField2 の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
連絡先の他の使用を目的として、連絡先の 2 番目のフィールドを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |dispidContactUserField2  <br/> |
|プロパティを設定します。  <br/> |PSETID_Address  <br/> |

|データを入力します。  <br/> |PT_UNICODE  <br/> |
