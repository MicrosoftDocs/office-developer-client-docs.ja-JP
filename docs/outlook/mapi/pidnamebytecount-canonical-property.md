# <a name="pidnamebytecount-canonical-property"></a>PidNameByteCount の標準的なプロパティ

  
  
**適用されます**: Outlook

|||
|:-----|:-----|
|フレンドリ名:  <br/> |なし  <br/> |
|プロパティを設定します。  <br/> |PS_PUBLIC_STRINGS  <br/> |
|プロパティ名:  <br/> |ByteCount  <br/> |
|データを入力します。  <br/> |PT_LONG  <br/> |
