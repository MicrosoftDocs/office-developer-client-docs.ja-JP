# <a name="pidtagspamjunksenders-canonical-property"></a>PidTagSpamJunkSenders 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
電子メール アドレスと受信拒否リストを構成するドメインのセミコロン区切りのリストが含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_SPAM_JUNK_SENDERS_W  <br/> |
|長い ID (LID):  <br/> |0x041A  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |スパム  <br/> |
