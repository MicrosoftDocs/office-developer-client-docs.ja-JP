# <a name="pidtagprofiletype-canonical-property"></a>PidTagProfileType 標準プロパティ

  
  
**適用対象**: Outlook 
  
インフォメーション ストアの種類が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_PROFILE_TYPE  <br/> |
|識別子:  <br/> |0x660A  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|領域:  <br/> |MAPI プロファイルの構成  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティの有効な値は次のとおりです。
  
- 1-プライマリ ユーザー
    
- 2: 代理人
    
- 3-パブリック ストア
    
- 4-定期売買
