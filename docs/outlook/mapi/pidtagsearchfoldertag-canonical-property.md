# <a name="pidtagsearchfoldertag-canonical-property"></a>PidTagSearchFolderTag 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
一致する検索フォルダーのコンテナーにこの定義のメッセージを同期に使用する値が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_WB_SF_TAG  <br/> |
|識別子:  <br/> |0x6847  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |検索  <br/> |
   
## <a name="remarks"></a>備考

定義のメッセージが変更されたとき、このプロパティが変更されます。 各イテレーションを変更する必要がありますが、それが一意でないです。
