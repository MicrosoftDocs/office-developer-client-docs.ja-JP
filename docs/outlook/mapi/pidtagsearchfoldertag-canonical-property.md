# <a name="pidtagsearchfoldertag-canonical-property"></a>PidTagSearchFolderTag 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
この定義メッセージと一致する検索フォルダーコンテナーを同期するために使用する値が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_WB_SF_TAG  <br/> |
|識別子:  <br/> |0x684 7  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |検索  <br/> |
   
## <a name="remarks"></a>解説

このプロパティは、定義メッセージが変更されたときに変更されます。 各反復処理を変更する必要がありますが、一意ではない場合があります。
