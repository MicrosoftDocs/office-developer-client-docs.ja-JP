# <a name="pidtagaddresstype-canonical-property"></a>PidTagAddressType 標準プロパティ

**適用されます**: Outlook 2013 |Outlook 2016 
  
SMTP など、メッセージング ユーザーの電子メール アドレスの種類が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_ADDRTYPE、PR_ADDRTYPE_A、PR_ADDRTYPE_W  <br/> |
|識別子:  <br/> |0x3002  <br/> |
|データの種類 :   <br/> |PT_UNICODE、PT_STRING8  <br/> |
|領域:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティでは、すべてのメッセージング ユーザーに共通の基本のアドレスのプロパティの例を示します。 どのメッセージング システムの特定のメッセージを処理するために MAPI を使用して指定します。
  
このプロパティは、 **PR_EMAIL_ADRESS** ([PidTagEmailAddress](pidtagemailaddress-canonical-property.md)) のアドレスの文字列の形式も決まります。 提供する文字列には、A Z からの大文字のアルファベット文字のみ、0 から 9 までの番号を含めることができます。
  
文字列の有効な例は次のとおりです。 
  
```cpp
FAX
MHS
PROFS
SMTP
X400

```
