# <a name="pidtagnondeliveryreportfromsearchkey-canonical-property"></a>PidTagNonDeliveryReportFromSearchKey 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
配信不能レポートに設定されている場合は、失敗した送信者ユーザーの検索キーを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_NDR_FROM_SEARCH_KEY  <br/> |
|識別子:  <br/> |0x6906  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |一般的なメッセージング  <br/> |
