# <a name="pidtagtelecommunicationsdevicefordeaftelephonenumber-canonical-property"></a>PidTagTelecommunicationsDeviceForDeafTelephoneNumber 標準プロパティ

  
  
**適用対象**: Outlook 
  
聴覚が不自由 (TTYTDD) の電話番号のメッセージング ユーザーの通信デバイスが含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_TTYTDD_PHONE_NUMBER、PR_TTYTDD_PHONE_NUMBER_A、PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|識別子:  <br/> |0x3A4B  <br/> |
|データの種類 :   <br/> |PT_STRING8、PT_UNICODE  <br/> |
|領域:  <br/> |MAPI メール ユーザー  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティでは、識別を提供し、メッセージングのユーザーに関する情報にアクセスします。 コンテンツは、メッセージングのユーザーとメッセージのユーザーの組織によって定義されます。
