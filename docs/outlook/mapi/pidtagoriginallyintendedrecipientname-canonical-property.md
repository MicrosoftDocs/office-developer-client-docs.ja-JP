# <a name="pidtagoriginallyintendedrecipientname-canonical-property"></a>PidTagOriginallyIntendedRecipientName の標準的なプロパティ

|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_ORIGINALLY_INTENDED_RECIPIENT_NAME  <br/> |
|識別子:  <br/> |0x0020  <br/> |
|データを入力します。  <br/> |PT_BINARY  <br/> |
