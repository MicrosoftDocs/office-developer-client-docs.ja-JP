# <a name="pidtagreferredbyname-canonical-property"></a>PidTagReferredByName 標準プロパティ

  
  
**適用対象**: Outlook 
  
メール ユーザーの参照の名前が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_REFERRED_BY_NAME、PR_REFERRED_BY_NAME_A、PR_REFERRED_BY_NAME_W  <br/> |
|識別子:  <br/> |0x3A47  <br/> |
|データの種類 :   <br/> |PT_STRING8、PT_UNICODE  <br/> |
|領域:  <br/> |MAPI メール ユーザー  <br/> |
