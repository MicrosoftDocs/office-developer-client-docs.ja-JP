# <a name="pidtaggivenname-canonical-property"></a>PidTagGivenName の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
1 つが含まれていますまたは受信者の名前を指定します。 
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_GIVEN_NAME、PR_GIVEN_NAME_A、PR_GIVEN_NAME_W  <br/> |
|識別子:  <br/> |0x3A06  <br/> |
|データを入力します。  <br/> |PT_STRING8、PT_UNICODE  <br/> |
|領域:  <br/> |MAPI メール ユーザー  <br/> |
   
## <a name="remarks"></a>備考

これらのプロパティでは、識別を提供し、受信者に関する情報にアクセスします。 受信者とその構造によって定義されます。
