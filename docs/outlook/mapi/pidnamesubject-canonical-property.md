# <a name="pidnamesubject-canonical-property"></a>PidNameSubject の標準的なプロパティ

  
  
**適用されます**: Outlook

|||
|:-----|:-----|
|フレンドリ名:  <br/> |なし  <br/> |
|プロパティを設定します。  <br/> |PS_PUBLIC_STRINGS  <br/> |
|プロパティ名:  <br/> |件名  <br/> |
|データを入力します。  <br/> |PT_UNICODE  <br/> |
|領域:  <br/> |Common  <br/> |
