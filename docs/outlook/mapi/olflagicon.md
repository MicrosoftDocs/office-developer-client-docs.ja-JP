# <a name="olflagicon"></a>OlFlagIcon

  
  
**適用されます**: Outlook 
  
アイテムのフラグを設定したかどうか、およびかどうかは、フラグが設定されたアイテムに関連付けられている色を指定します。
  
## <a name="quick-info"></a>クイック ヒント

```cpp
enum OlFlagIcon { 
    olNoFlagIcon=0, 
    olPurpleFlagIcon=1, 
    olOrangeFlagIcon=2, 
    olGreenFlagIcon=3, 
    olYellowFlagIcon=4, 
    olBlueFlagIcon=5, 
    olRedFlagIcon=6, 
}; 

```

## <a name="members"></a>メンバー

 _olNoFlagIcon_
  
> 項目に関連付けられているフラグはありません。
    
 _olPurpleFlagIcon_
  
> 紫は、フラグが設定されたアイテムに関連付けられて。
    
 _olOrangeFlagIcon_
  
> オレンジは、フラグが設定されたアイテムに関連付けられて。
    
 _olGreenFlagIcon_
  
> 緑は、フラグが設定されたアイテムに関連付けられて。
    
 _olYellowFlagIcon_
  
> 黄は、フラグが設定されたアイテムに関連付けられて。
    
 _olBlueFlagIcon_
  
> 青は、フラグの設定されたアイテムに関連付けられて。
    
 _olRedFlagIcon_
  
> 赤は、フラグの設定されたアイテムに関連付けられて。
