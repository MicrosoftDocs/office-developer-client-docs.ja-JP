# <a name="pidlidtoattendeesstring-canonical-property"></a>PidLidToAttendeesString 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
必須の出席者であるすべての送信可能な出席者の一覧が含まれる。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidToAttendeesString  <br/> |
|プロパティ セット:  <br/> |PSETID_Appointment  <br/> |
|長い ID (LID):  <br/> |0x0000823B  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |会議  <br/> |
   
## <a name="remarks"></a>注釈

各出席者の値は、PR_DISPLAY_NAME **の** アドレス帳のプロパティ ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)) です。 個別のエントリは、セミコロンで区切り、その後にスペースを指定する必要があります。 このプロパティは必須ではありません。
