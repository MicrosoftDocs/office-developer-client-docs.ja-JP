# <a name="pidtagotheraddresscountry-canonical-property"></a>PidTagOtherAddressCountry の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
メールにはユーザーの他の国/地域が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_OTHER_ADDRESS_COUNTRY、PR_OTHER_ADDRESS_COUNTRY_A、PR_OTHER_ADDRESS_COUNTRY_W  <br/> |
|識別子:  <br/> |0x3A60  <br/> |
|データを入力します。  <br/> |PT_UNICODE、PT_STRING8  <br/> |
|領域:  <br/> |アドレス帳  <br/> |
   
## <a name="remarks"></a>備考

これらのプロパティは、ユーザーまたはユーザーの組織によって定義されます。
