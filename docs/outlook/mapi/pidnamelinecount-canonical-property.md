# <a name="pidnamelinecount-canonical-property"></a>PidNameLineCount の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
ドキュメントに関連付けられているファイルの行の数を指定します。
  
|||
|:-----|:-----|
|フレンドリ名:  <br/> |なし  <br/> |
|プロパティを設定します。  <br/> |PS_PUBLIC_STRINGS  <br/> |
|プロパティ名:  <br/> |LineCount  <br/> |
|データを入力します。  <br/> |PT_LONG  <br/> |
|領域:  <br/> |Common  <br/> |
