# <a name="pidlidfreebusylocation-canonical-property"></a>PidLidFreeBusyLocation 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
クライアントが[[MS OXCICAL]](http://msdn.microsoft.com/library/a685a040-5b69-4c84-b084-795113fb4012%28Office.15%29.aspx)で指定したと、iCal ファイルとして連絡先の空き時間情報を取得できる URL パスを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidFreeBusyLocation  <br/> |
|プロパティを設定します。  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x000080D8  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|領域:  <br/> |Contact  <br/> |
