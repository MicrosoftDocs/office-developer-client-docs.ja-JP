# <a name="pidtagexceptionstarttime-canonical-property"></a>PidTagExceptionStartTime 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
例外が作成されると、開始の日付と時刻、コンピューターのローカル タイム ゾーンの例外を示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_EXCEPTION_STARTTIME  <br/> |
|識別子:  <br/> |0x7FFB  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |クラスによって定義されたメッセージの転送を可能な  <br/> |
   
## <a name="remarks"></a>備考

> [!NOTE]
> このプロパティは、情報提供する必要がありますがサポートされていない重要な情報にします。
