# <a name="pidtaginternetarticlenumber-canonical-property"></a>PidTagInternetArticleNumber 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージ ストア内のアイテムに関連付けられている数値です。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_INTERNET_ARTICLE_NUMBER  <br/> |
|識別子:  <br/> |0x0E23  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>備考

IMAPI ・ メッセージ ・ ストアでは、このプロパティは、IMAPI UID に対応します。
