**適用されます**: Outlook 
  
ストアのサポートされている機能を示すフラグが含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_CONTAB_STORE_SUPPORT_MASKS  <br/> |
|識別子:  <br/> |0x6621  <br/> |

|領域:  <br/> |連絡先のアドレス帳  <br/> |
   
## <a name="remarks"></a>備考

このプロパティは、連絡先フォルダーが含まれているストアから取得されます。 連絡先アドレス帳プロバイダーは、ストアのサポートされている機能の妥当性を評価するために使用します。 それは、アドレス帳の連絡先のプロファイル セクションのプロパティです。
