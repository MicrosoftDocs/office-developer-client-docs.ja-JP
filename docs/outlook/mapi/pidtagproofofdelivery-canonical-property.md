# <a name="pidtagproofofdelivery-canonical-property"></a>PidTagProofOfDelivery 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
このプロパティには、配信価値の ASN が1つ含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_PROOF_OF_DELIVERY  <br/> |
|識別子:  <br/> |0x0c11  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |MAPI 受信者  <br/> |
