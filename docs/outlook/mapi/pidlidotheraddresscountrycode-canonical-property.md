# <a name="pidlidotheraddresscountrycode-canonical-property"></a>PidLidOtherAddressCountryCode 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先のその他の住所の国コード部分を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidOtherAddressCountryCode  <br/> |
|プロパティセット:  <br/> |PSETID_Address  <br/> |
|ロング ID (LID):  <br/> |0x000080dc  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |連絡先  <br/> |
