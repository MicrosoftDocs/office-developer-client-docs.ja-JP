# <a name="pidtagautoforwardcomment-canonical-property"></a>PidTagAutoForwardComment 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016

|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_AUTO_FORWARD_COMMENT、PR_AUTO_FORWARD_COMMENT_A、PR_AUTO_FORWARD_COMMENT_W  <br/> |

|データの種類 :   <br/> |PT_STRING8、PT_UNICODE  <br/> |
