# <a name="pidlidcontactlinkname-canonical-property"></a>PidLidContactLinkName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
**dispidcontacts** ([PidLidContacts](pidlidcontacts-canonical-property.md)) プロパティの要素が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidcontactlinkname  <br/> |
|プロパティセット:  <br/> |PSETID_Common  <br/> |
|ロング ID (LID):  <br/> |0x00008586  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |連絡先  <br/> |
   
## <a name="remarks"></a>解説

**dispidcontactlinkname**プロパティの要素は、セミコロンとスペース (";") で区切られています。
