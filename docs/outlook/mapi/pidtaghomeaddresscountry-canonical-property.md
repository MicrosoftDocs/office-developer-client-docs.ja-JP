**適用対象**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_HOME_ADDRESS_COUNTRY、PR_HOME_ADDRESS_COUNTRY_A、PR_HOME_ADDRESS_COUNTRY_W  <br/> |
