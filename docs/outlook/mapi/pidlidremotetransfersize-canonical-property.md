# <a name="pidlidremotetransfersize-canonical-property"></a>PidLidRemoteTransferSize 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
リモートアイテムの転送サイズをバイト単位で示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidremotexフェリーサイズ  <br/> |
|プロパティセット:  <br/> |PSETID_Remote  <br/> |
|ロング ID (LID):  <br/> |0x00008f05  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |リモートメッセージ  <br/> |
