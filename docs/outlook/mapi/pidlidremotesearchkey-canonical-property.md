# <a name="pidlidremotesearchkey-canonical-property"></a>PidLidRemoteSearchKey 標準プロパティ

|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidremotesearchkey  <br/> |
|プロパティセット:  <br/> |PSETID_Remote  <br/> |
|ロング ID (LID):  <br/> |0x00008f06  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |リモートメッセージ  <br/> |
