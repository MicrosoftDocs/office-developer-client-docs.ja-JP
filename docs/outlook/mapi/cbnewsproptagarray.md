# <a name="cbnewsproptagarray"></a>CbNewSPropTagArray

  
  
**適用対象**: Outlook 
  
プロパティ タグの指定された番号を含む新しい[SPropTagArray](sproptagarray.md)構造体に割り当てられるバイト数を計算します。 
  
|||
|:-----|:-----|
|ヘッダー ファイル:  <br/> |Mapidefs.h  <br/> |
|関連の構造体。  <br/> |**SPropTagArray** <br/> |
   
```cpp
CbNewSPropTagArray (_ctag)
```

## <a name="parameters"></a>パラメーター

 __ctag_
  
> 新しい**SPropTagArray**構造体に含まれるプロパティ タグの数。
