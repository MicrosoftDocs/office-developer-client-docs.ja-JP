# <a name="pidtagconvertedeits-canonical-property"></a>PidTagConvertedEits 標準プロパティ

  
  
**適用対象**: Outlook 
  
変換後のメッセージ内のテキストの種類の識別子が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_CONVERTED_EITS  <br/> |
|識別子:  <br/> |0x000E  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|領域:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは、メッセージのテキスト要素に変換するエンコードされた情報の種類 (含む) が使用されたを示します。
