# <a name="pidlidclassificationkeep-canonical-property"></a>PidLidClassificationKeep 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
メッセージを転送または返信する場合に、メッセージ分類がメッセージと共に保持する必要があるかどうかを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidClassKeep  <br/> |
|プロパティを設定します。  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x000085BA  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|領域:  <br/> |メッセージ全般  <br/> |
   
## <a name="remarks"></a>注釈

詳細については、 [Outlook 2007 のメッセージ分類を展開する方法](http://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx)を参照してください。
