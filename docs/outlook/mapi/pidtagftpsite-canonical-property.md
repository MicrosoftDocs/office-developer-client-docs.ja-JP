# <a name="pidtagftpsite-canonical-property"></a>PidTagFtpSite 標準プロパティ

  
  
**適用対象**: Outlook 
  
連絡先のファイル転送プロトコル (FTP) の URL が含まれています。 FTP は、[RFC959] で指定されている、データを転送するために使用されるプロトコルです。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_FTP_SITE、PR_FTP_SITE_A、PR_FTP_SITE_W  <br/> |
|識別子:  <br/> |0x3A4C  <br/> |
|データの種類 :   <br/> |PT_STRING8、PT_UNICODE  <br/> |
|領域:  <br/> |MAPI メール ユーザー  <br/> |
