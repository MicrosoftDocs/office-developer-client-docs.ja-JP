# <a name="pidtaghasdeferredactionmessages-canonical-property"></a>PidTagHasDeferredActionMessages 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージに少なくとも1つの遅延アクションルールがある場合は、TRUE が含まれます。

|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |ルール  <br/> |
