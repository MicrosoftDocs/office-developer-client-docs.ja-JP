# <a name="pidlidcurrentversion-canonical-property"></a>PidLidCurrentVersion 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
アイテムの Outlook アプリケーションのビルド番号を表す長整数型 (long) の値を格納します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidCurrentVersion  <br/> |
|プロパティセット:  <br/> |PSETID_Common  <br/> |
|ロング ID (LID):  <br/> |0x00008552  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |一般的なメッセージング  <br/> |
