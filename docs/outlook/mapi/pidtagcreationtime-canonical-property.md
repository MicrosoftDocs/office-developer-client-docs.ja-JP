# <a name="pidtagcreationtime-canonical-property"></a>PidTagCreationTime 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージの作成日時が含まれます。 
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_CREATION_TIME  <br/> |
|識別子:  <br/> |0x3007  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |メッセージ時間  <br/> |
   
## <a name="remarks"></a>解説

メッセージストアは、作成する各メッセージに対してこのプロパティを設定します。
