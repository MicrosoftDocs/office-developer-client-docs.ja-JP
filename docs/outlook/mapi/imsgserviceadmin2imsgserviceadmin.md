## <a name="remarks"></a>注釈

**IMsgServiceAdmin2**インターフェイスは、 [IMsgServiceAdmin](imsgserviceadminiunknown.md)インターフェイスを公開するのと同じオブジェクトによって公開されており、Microsoft outlook 2003 以降の outlook の MAPI サブシステムの実装を使用して利用できるようになっています。
