# <a name="pidtagrecipientstatus-canonical-property"></a>PidTagRecipientStatus 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
トランスポート プロバイダー間で配信責任を割り当てる場合に MAPI スプーラーによって使用される値が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_RECIPIENT_STATUS  <br/> |
|識別子:  <br/> |0x0E15  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |MAPI 送信不可  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティを使用しません。 MAPI で使用するために予約されています。
