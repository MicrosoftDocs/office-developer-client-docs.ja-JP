# <a name="cbnewspropproblemarray"></a>CbNewSPropProblemArray

  
  
**適用対象**: Outlook 
  
[SPropProblem](spropproblem.md)構造体の指定した番号を含む新しい[SPropProblemArray](spropproblemarray.md)構造体に割り当てられるバイト数を計算します。 
  
|||
|:-----|:-----|
|ヘッダー ファイル:  <br/> |Mapidefs.h  <br/> |
|関連の構造体。  <br/> |**SPropProblemArray** <br/> |
   
```cpp
CbNewSPropProblemArray (_cprob)
```

## <a name="parameters"></a>パラメーター

 __cprob_
  
> 新しい**SPropProblemArray**構造体に含まれる**SPropProblem**構造体の数です。
