# <a name="cbnewspropproblemarray"></a>CbNewSPropProblemArray

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
指定した数の[sprop問題](spropproblem.md)構造を含む新しい[sprop問題の配列](spropproblemarray.md)構造に割り当てるバイト数を計算します。 
  
|||
|:-----|:-----|
|ヘッダー ファイル:  <br/> |mapidefs.h  <br/> |
|関連する構造:  <br/> |**SPropProblemArray** <br/> |
   
```cpp
CbNewSPropProblemArray (_cprob)
```

## <a name="parameters"></a>パラメーター

 __cprob_
  
> 新しい**spropproblem の配列**構造に含めることができる**spropproblem**の構造体の数。
