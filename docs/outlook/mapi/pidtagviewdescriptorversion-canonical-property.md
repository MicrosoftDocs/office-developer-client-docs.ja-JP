# <a name="pidtagviewdescriptorversion-canonical-property"></a>PidTagViewDescriptorVersion 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ビュー記述子のバージョンを示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_VD_VERSION  <br/> |
|識別子:  <br/> |0x7007  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |その他  <br/> |
   
## <a name="remarks"></a>解説

このプロパティの値は8である必要があります。
