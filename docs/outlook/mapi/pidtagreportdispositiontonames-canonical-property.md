# <a name="pidtagreportdispositiontonames-canonical-property"></a>PidTagReportDispositionToNames 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
受信確認を要求するメッセージのレポート受信者の名前を示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_REPORT_DISPOSITION_TO_NAMES、PR_REPORT_DISPOSITION_TO_NAMES_A、PR_REPORT_DISPOSITION_TO_NAMES_W  <br/> |
|識別子:  <br/> |0x0083  <br/> |
|データの種類 :   <br/> |PT_MV_STRING8、PT_UNICODE  <br/> |
|エリア:  <br/> |MAPI エンベロープ  <br/> |
