# <a name="mapi-file-formats"></a>MAPI ファイル形式

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
MAPI は、クライアントとサービスプロバイダーがメッセージサービスとそのサービスプロバイダおよびフォームを記述するために使用するさまざまな構成ファイルの形式を公開します。
  
次の表で、これらのファイルを名前または種類別に説明します。
  
|**構成ファイル**|**説明**|
|:-----|:-----|
|mapisvc.inf  <br/> |MAPI サブシステム、メッセージサービス、およびサービスプロバイダーの構成情報が含まれています。  <br/> |
|拡張子が cfg のファイル  <br/> |フォームの構成情報が含まれています。  <br/> |
   
mapisvc.inf のファイル形式の詳細については、「 [mapisvc.inf のファイル形式](file-format-of-mapisvc-inf.md)」を参照してください。 
  
フォーム構成ファイルのファイル形式の詳細については、「[フォーム構成ファイルのファイル形式](file-format-of-form-configuration-files.md)」を参照してください。
