# <a name="mapi-file-formats"></a>MAPI ファイル形式

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
MAPI は、メッセージ サービスと、サービス プロバイダー、およびフォームを記述するクライアントとサービス ・ プロバイダーを使用するさまざまな構成ファイルの形式を公開します。
  
次の表では、これらのファイルの名前または種類をについて説明します。
  
|**構成ファイル**|**説明**|
|:-----|:-----|
|MapiSvc.inf  <br/> |MAPI サブシステム、メッセージ サービス、およびサービス プロバイダーの構成情報が含まれています。  <br/> |
|.Cfg ファイルの拡張子を持つファイル  <br/> |フォームの構成情報が含まれています。  <br/> |
   
MapiSvc.inf のファイル形式の詳細については、 [MapiSvc.inf のファイル形式](file-format-of-mapisvc-inf.md)を参照してください。 
  
フォーム構成ファイルのファイル形式の詳細については、[フォーム構成ファイルの形式のファイル](file-format-of-form-configuration-files.md)を参照してください。
