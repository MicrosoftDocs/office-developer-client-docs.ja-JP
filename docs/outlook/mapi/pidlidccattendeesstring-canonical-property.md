**適用されます**: Outlook 
  
出席者全員のうっかりしているユーザーも任意出席者の一覧が含まれています。

## <a name="remarks"></a>備考

各出席者の値は、出席者のアドレス帳の**PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)) プロパティです。 別のエントリは、スペースの後にセミコロンで区切る必要があります。 このプロパティが必要ではありません。
