# <a name="pidlidcontactuserfield4-canonical-property"></a>PidLidContactUserField4 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先のその他の使用を目的とした、連絡先の4番目のフィールドを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidContactUserField4  <br/> |
|プロパティセット:  <br/> |PSETID_Address  <br/> |
|ロング ID (LID):  <br/> |0x00008052  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |連絡先  <br/> |
   
## <a name="remarks"></a>解説

このプロパティには、他の連絡先固有のプロパティとは無関係のテキストが含まれています。
