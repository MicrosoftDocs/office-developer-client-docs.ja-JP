# <a name="pidlidcontactuserfield4-canonical-property"></a>PidLidContactUserField4 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の他の使用を目的として、取引先担当者の 4 番目のフィールドを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidContactUserField4  <br/> |
|プロパティを設定します。  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x00008052  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>備考

このプロパティには、他の連絡先に固有のプロパティに関連するテキストが含まれています。
