# <a name="pidlidsharingresponsetime-canonical-property"></a>PidLidSharingResponseTime 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
共有要求の受信者が共有応答を送信する時刻を指定します。 これは、共有メッセージのプロパティです。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidSharingResponseTime  <br/> |
|プロパティを設定します。  <br/> |PSETID_Sharing  <br/> |
|長い ID (LID):  <br/> |0x00008A28  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|領域:  <br/> |共有  <br/> |
