# <a name="pidlidnoteheight-canonical-property"></a>PidLidNoteHeight 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
表示されているメッセージ ウィンドウの高さをピクセル単位で指定します。 
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidNoteHeight  <br/> |
|プロパティを設定します。  <br/> |PSETID_Note  <br/> |
|長い ID (LID):  <br/> |0x00008B03  <br/> |
|データの種類 :   <br/> |PT_LONG  <br/> |
|エリア:  <br/> |付箋  <br/> |
   
## <a name="remarks"></a>備考

この値は 0 より大きくする必要があります。
