# <a name="pidtagabdefaultpab-canonical-property"></a>PidTagAbDefaultPab の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
個人用アドレス帳 (PAB) として使用するアドレス帳コンテナーのエントリの識別子が含まれています。 
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_AB_DEFAULT_PAB  <br/> |
|識別子:  <br/> |0x3D07  <br/> |
|データを入力します。  <br/> |PT_BINARY  <br/> |
|領域:  <br/> |アドレス帳  <br/> |
   
## <a name="remarks"></a>備考

このプロパティを使用しません。 MAPI によって使用するために予約されています。
