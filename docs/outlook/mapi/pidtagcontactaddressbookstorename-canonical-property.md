# <a name="pidtagcontactaddressbookstorename-canonical-property"></a>PidTagContactAddressBookStoreName の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
アドレス帳プロバイダーに関連付けられているストアの表示名が含まれています。 
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_CONTAB_STORE_NAME、PR_CONTAB_STORE_NAME_A、PR_CONTAB_STORE_NAME_W  <br/> |
|識別子:  <br/> |0x6612  <br/> |
|データを入力します。  <br/> |PT_UNICODE、PT_STRING8  <br/> |
|領域:  <br/> |連絡先のアドレス帳  <br/> |
   
## <a name="remarks"></a>備考

これらのプロパティは、連絡先のアドレス帳コンテナー、および連絡先のアドレス帳コンテナーのテーブル内の列には。
