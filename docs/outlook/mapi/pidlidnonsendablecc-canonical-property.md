# <a name="pidlidnonsendablecc-canonical-property"></a>PidLidNonSendableCc 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
すべての連絡先である出席者も任意出席者の一覧が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidNonSendableCC  <br/> |
|プロパティを設定します。  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x00008537  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|領域:  <br/> |会議  <br/> |
   
## <a name="remarks"></a>注釈

各出席者の値は、出席者のアドレス帳の**PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)) プロパティです。 別のエントリは、スペースの後にセミコロンで区切る必要があります。 このプロパティが必要ではありません。
