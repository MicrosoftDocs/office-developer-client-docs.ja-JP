# <a name="pidnamexsharingprovidername-canonical-property"></a>PidNameXSharingProviderName 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
**DispidSharingProviderGuid** ([PidLidSharingProviderGuid](pidlidsharingproviderguid-canonical-property.md)) のプロパティで指定された共有プロバイダーの表示名を表します。
  
|||
|:-----|:-----|
|フレンドリ名:  <br/> |なし  <br/> |
|プロパティを設定します。  <br/> |PS_INTERNET_HEADERS  <br/> |
|プロパティ名:  <br/> |X の共有プロバイダーの名  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|領域:  <br/> |共有  <br/> |
