# <a name="pidtaglocaleid-canonical-property"></a>PidTagLocaleId の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
翻訳システムにより生成されたメッセージは、配信レポートなどの言語ロケールを設定します。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_LOCALE_ID  <br/> |
|識別子:  <br/> |0x66A1  <br/> |
|データを入力します。  <br/> |PT_LONG  <br/> |
|領域:  <br/> |その他  <br/> |
