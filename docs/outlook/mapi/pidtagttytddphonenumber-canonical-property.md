# <a name="pidtagttytddphonenumber-canonical-property"></a>PidTagTtyTddPhoneNumber の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
連絡先のテキスト電話 (TTY) の電話番号または、聴覚が不自由 (TDD) の電気通信デバイスが含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_TTYTDD_PHONE_NUMBER、PR_TTYTDD_PHONE_NUMBER_A、PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|識別子:  <br/> |0x3A4B  <br/> |
|データを入力します。  <br/> |PT_STRING8、PT_UNICODE  <br/> |
|領域:  <br/> |連絡先  <br/> |
