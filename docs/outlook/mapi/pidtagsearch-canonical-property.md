# <a name="pidtagsearch-canonical-property"></a>PidTagSearch 標準プロパティ

  
  
**適用対象**: Outlook 
  
高度な検索に使用されるコンテナー オブジェクトが含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_SEARCH  <br/> |
|識別子:  <br/> |0x3607  <br/> |
|データの種類 :   <br/> |PT_OBJECT  <br/> |
|領域:  <br/> |アドレス帳コンテナー  <br/> |
   
## <a name="remarks"></a>注釈

プロパティを提供する高度な検索機能をサポートしていないコンテナーがありませんでした。
