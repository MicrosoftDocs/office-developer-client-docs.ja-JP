# <a name="pidtaglastverbexecutiontime-canonical-property"></a>PidTagLastVerbExecutionTime 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
最後の動詞が実行された時刻を格納します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_LAST_VERB_EXECUTION_TIME  <br/> |
|識別子:  <br/> |0x1082  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |履歴  <br/> |
