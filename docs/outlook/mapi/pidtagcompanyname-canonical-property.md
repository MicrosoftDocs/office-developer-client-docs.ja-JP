# <a name="pidtagcompanyname-canonical-property"></a>PidTagCompanyName 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
受信者の会社名が含まれます。 
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_COMPANY_NAME、PR_COMPANY_NAME_A、PR_COMPANY_NAME_W  <br/> |
|識別子:  <br/> |0x3a16  <br/> |
|データの種類 :   <br/> |PT_UNICODE、PT_STRING8  <br/> |
|エリア:  <br/> |連絡先  <br/> |
   
## <a name="remarks"></a>解説

これらのプロパティは、受信者に関する id とアクセス情報を提供します。 受信者と受信者の組織によって定義されます。
