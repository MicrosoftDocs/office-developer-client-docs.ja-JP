# <a name="pidtagx400contenttype-canonical-property"></a>PidTagX400ContentType 標準プロパティ

  
  
**適用対象**: Outlook 
  
このプロパティには、送信されたメッセージのコンテンツ タイプが含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_X400_CONTENT_TYPE  <br/> |
|識別子:  <br/> |0x003C  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|領域:  <br/> |X400  <br/> |
