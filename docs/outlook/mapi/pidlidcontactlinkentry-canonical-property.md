# <a name="pidlidcontactlinkentry-canonical-property"></a>PidLidContactLinkEntry 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
このメッセージ オブジェクトによってリンクされた **アドレス帳エントリの** 一覧を格納します。 
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidContactLinkEntry  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x00008585  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |Contact  <br/> |
