# <a name="pidtagotheraddresspostofficebox-canonical-property"></a>PidTagOtherAddressPostOfficeBox 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
連絡先が他のアドレスのための私書箱が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_OTHER_ADDRESS_POST_OFFICE_BOX、PR_OTHER_ADDRESS_POST_OFFICE_BOX_A、PR_OTHER_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|識別子:  <br/> |0x3A64  <br/> |
|データの種類 :   <br/> |PT_UNICODE、PT_STRING8  <br/> |
|領域:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティは、ユーザーまたはユーザーの組織によって定義されます。
