# <a name="mapi-form-libraries"></a>MAPI フォーム ライブラリ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
フォームのサーバーを整理しをクライアント アプリケーションに簡単に使用できるようにするために、MAPI フォームのアーキテクチャには、フォーム ライブラリには、インストール、管理、レプリケーション、およびフォームのサーバーの使用をサポートが含まれています。
  
フォーム ライブラリは、フォームのサーバーとサーバーのフォームを読み込むときに、フォーム マネージャーを使用していることについての構成情報のリポジトリです。 
  
> [!NOTE]
> MAPI の履歴、早い段階では、フォーム ライブラリが」の「レジストリ。 を呼び出されました。 このためは、「レジストリ」という用語がいくつかフォームに関連する MAPI インターフェイス、プロパティ、構成ファイルのエントリなどで永続化されます。 アプリケーションのインターフェイスは、これらの条件のいずれかの使用を必要とする場合は「ライブラリ」という用語排他的に使用する必要があります。
