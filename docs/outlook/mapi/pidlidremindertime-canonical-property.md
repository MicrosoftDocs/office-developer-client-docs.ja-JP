# <a name="pidlidremindertime-canonical-property"></a>PidLidReminderTime 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
アラームの初期信号時間を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidReminderTime  <br/> |
|プロパティ セット:  <br/> |PSETID_Common  <br/> |
|長い ID (LID):  <br/> |0x00008502  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |Reminder  <br/> |
   
## <a name="remarks"></a>注釈

予定表オブジェクトの場合、このプロパティは、予定の開始時刻であるユーザーが遅れた時刻を表します。 クライアントは、協定世界時 (UTC) で値を設定する必要があります。
