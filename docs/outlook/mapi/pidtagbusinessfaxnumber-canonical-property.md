# <a name="pidtagbusinessfaxnumber-canonical-property"></a>PidTagBusinessFaxNumber 標準プロパティ

  
  
**適用対象**: Outlook 
  
受信者のビジネスの fax の電話番号が含まれています。 
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_BUSINESS_FAX_NUMBER、PR_BUSINESS_FAX_NUMBER_A、PR_BUSINESS_FAX_NUMBER_W  <br/> |
|識別子:  <br/> |0x3A24  <br/> |
|データの種類 :   <br/> |PT_UNICODE、PT_STRING8  <br/> |
|領域:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティでは、id、および受信者に関する情報にアクセスを提供するプロパティの例を示します。 これらのプロパティは、受信者と受信者の組織によって定義されます。
