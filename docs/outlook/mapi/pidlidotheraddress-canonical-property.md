# <a name="pidlidotheraddress-canonical-property"></a>PidLidOtherAddress 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の他のアドレスの完全なアドレスを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidOtherAddress  <br/> |
|プロパティ セット:  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x0000801C  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注釈

このプロパティは、他の物理アドレス プロパティの組み合わせで、クライアント ロケールに基づく必要があります。
