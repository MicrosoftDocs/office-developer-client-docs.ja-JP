# <a name="pidtagradiotelephonenumber-canonical-property"></a>PidTagRadioTelephoneNumber 標準プロパティ

  
  
**適用されます**: Outlook 2013 |Outlook 2016 
  
受信者の携帯電話番号が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_RADIO_TELEPHONE_NUMBER、PR_RADIO_TELEPHONE_NUMBER_A、PR_RADIO_TELEPHONE_NUMBER_W  <br/> |
|識別子:  <br/> |0x3A1D  <br/> |
|データの種類 :   <br/> |PT_STRING8、PT_UNICODE  <br/> |
|領域:  <br/> |MAPI メール ユーザー  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティでは、識別を提供し、受信者の情報にアクセスします。 受信者と受信者の組織によって定義されます。
