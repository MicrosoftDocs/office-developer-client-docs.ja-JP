# <a name="pidlidoldwhenendwhole-canonical-property"></a>PidLidOldWhenEndWhole 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
会議の更新前の**dispidApptEndWhole** ([PidLidAppointmentEndWhole](pidlidappointmentendwhole-canonical-property.md)) プロパティの元の値を示します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidoldwhenendwhole  <br/> |
|プロパティセット:  <br/> |PSETID_Meeting  <br/> |
|ロング ID (LID):  <br/> |0x0000002a  <br/> |
|データの種類 :   <br/> |PT_SYSTIME  <br/> |
|エリア:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>解説

このプロパティは必須ではありません。
