# <a name="pidnametitle-canonical-property"></a>PidNameTitle 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
ドキュメントに関連付けられているファイルのタイトルを指定します。
  
|||
|:-----|:-----|
|フレンドリ名:  <br/> |なし  <br/> |
|プロパティを設定します。  <br/> |PS_PUBLIC_STRINGS  <br/> |
|プロパティ名:  <br/> |Title  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Common  <br/> |
