# <a name="pidlidagingdontageme-canonical-property"></a>PidLidAgingDontAgeMe 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
アイテムが自動的にアーカイブされない場合は TRUE の値を指定します。

|ロング ID (LID):  <br/> |0x0000850e  <br/> |
|データの種類 :   <br/> |PT_BOOLEAN  <br/> |
|エリア:  <br/> |共通  <br/> |
