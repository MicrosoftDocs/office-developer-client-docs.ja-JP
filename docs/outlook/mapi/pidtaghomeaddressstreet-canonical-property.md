# <a name="pidtaghomeaddressstreet-canonical-property"></a>PidTagHomeAddressStreet 標準プロパティ

  
  
**適用対象**: Outlook 
  
ユーザーの住所の番地の部分が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_HOME_ADDRESS_STREET、PR_HOME_ADDRESS_STREET_A、PR_HOME_ADDRESS_STREET_W  <br/> |
|識別子:  <br/> |0x3A5D  <br/> |
|データの種類 :   <br/> |PT_STRING8、PT_UNICODE  <br/> |
|領域:  <br/> |MAPI メール ユーザー  <br/> |
   
## <a name="remarks"></a>注釈

これらのプロパティは、ユーザーまたはユーザーの組織によって定義されます。
