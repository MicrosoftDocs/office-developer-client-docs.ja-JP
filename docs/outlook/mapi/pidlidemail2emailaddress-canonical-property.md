# <a name="pidlidemail2emailaddress-canonical-property"></a>PidLidEmail2EmailAddress 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
連絡先の 2 番目の電子メール アドレスを指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidEmail2EmailAddress  <br/> |
|プロパティを設定します。  <br/> |PSETID_Address  <br/> |
|長い ID (LID):  <br/> |0x00008093  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>備考

このプロパティの値は、この電子メール アドレスに指定されたアドレスの種類に適したである必要があります。
