# <a name="pidtaghtml-canonical-property"></a>PidTagHtml 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
HTML 形式でメッセージ本文のテキストが含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |PR_HTML  <br/> |
|識別子:  <br/> |0x1013  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |メッセージ全般  <br/> |
