# <a name="pidlidreminderfileparameter-canonical-property"></a>PidLidReminderFileParameter 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
そのオブジェクトのアラームが期限切れになったときにクライアントが再生するサウンドのファイル名を指定します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidReminderFileParam  <br/> |
|プロパティセット:  <br/> |PSETID_Common  <br/> |
|ロング ID (LID):  <br/> |0x0000851f  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |Reminder  <br/> |
   
## <a name="remarks"></a>解説

このプロパティが存在しない場合、クライアントは既定値を使用できます。
