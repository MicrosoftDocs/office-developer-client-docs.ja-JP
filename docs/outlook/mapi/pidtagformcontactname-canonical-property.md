# <a name="pidtagformcontactname-canonical-property"></a>PidTagFormContactName の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
フォームに関する情報の連絡先の名前が含まれています。 
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_FORM_CONTACT_NAME、PR_FORM_CONTACT_NAME_A、PR_FORM_CONTACT_NAME_W  <br/> |
|識別子:  <br/> |0x3303  <br/> |
|データを入力します。  <br/> |PT_STRING8、PT_UNICODE  <br/> |
|領域:  <br/> |一般的な MAPI  <br/> |
   
## <a name="remarks"></a>備考

連絡先には、通常、人や、フォームの管理を担当するエイリアスの名前が含まれます。
