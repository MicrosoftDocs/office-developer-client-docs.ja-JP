# <a name="pidlidtaskowner-canonical-property"></a>PidLidTaskOwner 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
タスクの所有者の名前が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidTaskOwner  <br/> |
|プロパティを設定します。  <br/> |PSETID_Task  <br/> |
|長い ID (LID):  <br/> |0x0000811F  <br/> |
|データの種類 :   <br/> |PT_UNICODE  <br/> |
|エリア:  <br/> |タスク  <br/> |
