# <a name="pidlidscriptstream-canonical-property"></a>PidLidScriptStream 標準プロパティ

  
  
**適用対象**: Outlook 2013 | Outlook 2016 
  
メッセージに関連付けられたユーザー設定フォームの背後にある、Visual Basic Scripting Edition (VBScript) で記述されたコードを表します。
  
|||
|:-----|:-----|
|関連するプロパティ:  <br/> |dispidscriptstream  <br/> |
|プロパティセット:  <br/> |PSETID_Common  <br/> |
|ロング ID (LID):  <br/> |0x00008541  <br/> |
|データの種類 :   <br/> |PT_BINARY  <br/> |
|エリア:  <br/> |実行時の構成  <br/> |
