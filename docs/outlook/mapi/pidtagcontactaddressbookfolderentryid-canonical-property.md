# <a name="pidtagcontactaddressbookfolderentryid-canonical-property"></a>PidTagContactAddressBookFolderEntryId の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
メール ユーザーが保存されているメッセージ ストアの連絡先フォルダーの識別子が含まれています。 
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_CONTAB_FOLDER_ENTRYID  <br/> |
|識別子:  <br/> |0x6610  <br/> |
|データを入力します。  <br/> |PT_BINARY  <br/> |
|領域:  <br/> |連絡先のアドレス帳  <br/> |
   
## <a name="remarks"></a>備考
