# <a name="pidtagcontactaddressbookuid-canonical-property"></a>PidTagContactAddressBookUid の標準的なプロパティ

  
  
**適用されます**: Outlook 
  
連絡先アドレス帳には、このセッションを識別する UID が含まれています。
  
|||
|:-----|:-----|
|関連するプロパティ。  <br/> |PR_CONTAB_UID  <br/> |
|識別子:  <br/> |0x6601  <br/> |
|データを入力します。  <br/> |PT_BINARY  <br/> |
|領域:  <br/> |連絡先のアドレス帳  <br/> |
   
## <a name="remarks"></a>備考

構成されている複数の連絡先アドレス帳を使っている場合、それぞれのこのプロパティを設定する必要があります。
