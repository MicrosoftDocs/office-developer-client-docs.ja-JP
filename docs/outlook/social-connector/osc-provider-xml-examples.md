# <a name="osc-provider-xml-examples"></a>.osc プロバイダーの XML の例

このセクションには、Outlook Social Connector (.osc) プロバイダーの xml スキーマに準拠する**機能**、 **friends**、および**activityfeed** xml 要素を使用する xml の例が記載されています。 
  
- [capabilities xml の例](capabilities-xml-example.md)は、.osc プロバイダーの機能と要件を指定する、.osc プロバイダーの**機能**xml を示しています。 
    
- [friends xml の例](friends-xml-example.md)は、ソーシャルネットワーク上の2人の人々の**フレンド**xml を示しています。 
    
- [アクティビティフィード xml の例](activity-feed-xml-example.md)は、2つのプロファイル画像の更新、状態の更新、およびブログ投稿の4つのアクティビティの**activityfeed** xml を示しています。 また、XML は、対応するアクティビティを表示するための3つのアクティビティ表示テンプレートも指定します。
