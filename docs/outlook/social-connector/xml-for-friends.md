<span data-ttu-id="3767a-104">Microsoft  Outlook Social Connector (OSC) プロバイダー XML スキーマの friends 要素を使用すると、OSC プロバイダーは、ソーシャル ネットワーク内の Outlook ユーザーに関連付けられているユーザーの一覧に関する情報を指定できます。</span><span class="sxs-lookup"><span data-stu-id="3767a-104">The **friends** element in the Microsoft Outlook Social Connector (OSC) provider XML schema allows an OSC provider to specify information for a list of persons associated with an Outlook user in the social network.</span></span> <span data-ttu-id="3767a-105">OSC プロバイダーがキャッシュ同期をサポートしている場合、このユーザーの一覧には、ソーシャル ネットワーク上のユーザー Outlookのフレンドだけが含まれる。</span><span class="sxs-lookup"><span data-stu-id="3767a-105">If the OSC provider supports cached synchronization, this list of person will contain only friends of the Outlook user on the social network.</span></span> <span data-ttu-id="3767a-106">OSC がオンデマンド同期またはハイブリッド同期をサポートしている場合、このリストには、ユーザーのフレンドとフレンド以外のOutlookがあります。</span><span class="sxs-lookup"><span data-stu-id="3767a-106">If the OSC supports on-demand or hybrid synchronization, this list may contain both friends and non-friends of the Outlook user.</span></span>
