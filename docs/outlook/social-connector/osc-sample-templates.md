# <a name="osc-sample-templates"></a>.osc サンプルテンプレート

Outlook Social Connector 2013 プロバイダリファレンスは、C++、C#、Visual Basic のいくつかのプロバイダーテンプレートです。 テンプレートは、 [Outlook Social Connector 2013: プロバイダテンプレート](https://code.msdn.microsoft.com/Outlook-Social-Connector-73fd8d2c)からダウンロードできます。
  
プロバイダーテンプレートは、プロバイダーを任意の言語で構築するときに使用できます。 プロバイダーのダウンロードサイズを最小にし、プロバイダーの負荷時間を最小限に抑えるには、C++ を使用して運用プロバイダーを構築する必要があります。 .net framework 3.5 SP1 またはそれ以降のバージョンの .net framework をコンピューターにインストールしているユーザーのプロバイダーを開発している場合は、C# または Visual Basic を使用してマネージプロバイダーを記述することも検討できます。
