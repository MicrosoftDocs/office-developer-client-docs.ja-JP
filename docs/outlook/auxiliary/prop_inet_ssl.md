# <a name="propinetssl"></a>PROP_INET_SSL

一般的なインターネット メールボックスのセキュア ソケット レイヤー (SSL) を使用するかどうかを指定します。
  
## <a name="quick-info"></a>クイック ヒント

|||
|:-----|:-----|
|識別子:  <br/> |0x0105  <br/> |
|プロパティの種類:  <br/> |PT_DWORD  <br/> |
|プロパティ タグ。  <br/> |0x01050003  <br/> |
|アクセス:  <br/> |値の取得のみ可能です。  <br/> |
   
## <a name="remarks"></a>備考

ゼロの値は SSL を使用しない、それ以外の場合、SSL を使用します。
