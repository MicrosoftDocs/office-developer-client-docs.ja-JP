# <a name="iolkenumreset"></a>IOlkEnum::Reset

先頭に列挙子をリセットします。
  
## <a name="quick-info"></a>クイック ヒント

[IOlkEnum](iolkenum.md)を参照してください。
  
```cpp
HRESULT IOlkEnum::Reset ();
```

## <a name="return-values"></a>戻り値

呼び出しが成功した場合は S_OKそれ以外の場合はエラー コードです。
