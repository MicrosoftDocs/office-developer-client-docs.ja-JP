# <a name="propinetport"></a>PROP_INET_PORT

一般的なインターネットメールボックスのポート番号を表します。
  
## <a name="quick-info"></a>クイック ヒント

|||
|:-----|:-----|
|識別子:  <br/> |0x0104  <br/> |
|プロパティの種類:  <br/> |PT_DWORD  <br/> |
|プロパティタグ:  <br/> |0x01040003  <br/> |
|接続  <br/> |読み取り専用  <br/> |
   
## <a name="remarks"></a>解説

たとえば、POP アカウントの場合は110です。
