# <a name="prop_inet_port"></a>PROP_INET_PORT

一般的なインターネット メールボックスのポート番号を表します。
  
## <a name="quick-info"></a>クイック ヒント

|||
|:-----|:-----|
|識別子:  <br/> |0x0104  <br/> |
|プロパティの種類:  <br/> |PT_DWORD  <br/> |
|プロパティ タグ:  <br/> |0x01040003  <br/> |
|アクセス:  <br/> |読み取り専用  <br/> |
   
## <a name="remarks"></a>注釈

たとえば、POP アカウントの場合は 110 です。
