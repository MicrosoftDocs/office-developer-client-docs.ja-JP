# <a name="propsmtpuseauth"></a>PROP_SMTP_USE_AUTH

SMTP アカウントの認証を使用するかどうかを指定します。
  
## <a name="quick-info"></a>クイック ヒント

|||
|:-----|:-----|
|識別子:  <br/> |0x0203  <br/> |
|プロパティの種類:  <br/> |PT_DWORD  <br/> |
|プロパティタグ:  <br/> |0x02030003  <br/> |
|接続  <br/> |読み取り専用  <br/> |
   
## <a name="remarks"></a>解説

ゼロ値は認証を使用しないことを意味します。それ以外の場合は認証を使用します。
