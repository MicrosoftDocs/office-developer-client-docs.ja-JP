# <a name="about-the-data-degradation-layer-api"></a>データ変換レイヤー API について

データ低下層 API は、Outlook オブジェクトの基になるネイティブ文字形式を非表示にする機能を提供します。これにより、クライアントは、オブジェクトのネイティブ文字形式ではなく、適切な文字形式でオブジェクトにアクセスできるようになります。 [定数 (データ低下レイヤー API)](constants-data-degradation-layer-api.md)と、指定された形式でラップされた Outlook オブジェクトを返す関数[HrCreateNewWrappedObject](hrcreatenewwrappedobject.md)の定数定義が提供されます。
