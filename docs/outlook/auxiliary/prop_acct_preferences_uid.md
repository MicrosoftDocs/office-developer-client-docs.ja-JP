# <a name="propacctpreferencesuid"></a>PROP_ACCT_PREFERENCES_UID

アカウントの設定を格納するプロファイルセクションの一意識別子 (UID) を取得します。 
  
## <a name="quick-info"></a>クイック ヒント

[IOlkAccount](iolkaccount.md)を参照してください。
  
|||
|:-----|:-----|
|識別子:  <br/> |0x0022  <br/> |
|プロパティの種類:  <br/> |PT_BINARY  <br/> |
|プロパティタグ:  <br/> |0x00220102  <br/> |
|接続  <br/> |読み取り専用  <br/> |
   
## <a name="remarks"></a>解説

**PROP_ACCT_PREFERENCES_UID**を[imapisupport::](https://msdn.microsoft.com/library/cd1fa994-9531-46c4-94e5-505e7f90b884%28Office.15%29.aspx) openprofilesection の呼び出しで使用して、アカウントの設定を含むプロファイルセクションを取得します。
