## <a name="remarks"></a>解説

このインターフェイスのメンバーのほとんどは、Outlook の内部使用のために予約されたプレースホルダーであり、変更される可能性があります。 空き時間情報プロバイダーは、指定された値のみを実装する必要があり、指定された戻り値のみを返します。
