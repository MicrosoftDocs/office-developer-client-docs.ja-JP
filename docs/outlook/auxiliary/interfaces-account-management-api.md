# <a name="interfaces-account-management-api"></a>インタ フェース (アカウント管理 API)

このセクションでは、アカウント管理 API のインターフェイスについて説明します。
  
|**インターフェイス**|**説明**|
|:-----|:-----|
|[IOlkAccount](iolkaccount.md) <br/> |取得および設定のプロパティおよびその他のアカウント情報をサポートします。  <br/> |
|[IOlkAccountHelper](iolkaccounthelper.md) <br/> |アカウントを管理する現在の MAPI セッションでは、ヘルパー機能を提供します。  <br/> |
|[IOlkAccountManager](iolkaccountmanager.md) <br/> |アカウントへのアクセスを管理し、アカウントの変更に関する通知を設定します。  <br/> |
|[IOlkAccountNotify](iolkaccountnotify.md) <br/> |アカウントへの変更をクライアントにコールバックを提供します。  <br/> |
|[IOlkEnum](iolkenum.md) <br/> |[IUnknown](http://msdn.microsoft.com/library/com.iunknown%28Office.15%29.aspx)オブジェクトとしてアカウントの列挙をサポートします。  <br/> |
|[IOlkErrorUnknown](iolkerrorunknown.md) <br/> |最後のエラーに関する追加情報を提供します。  <br/> |
