# <a name="propinetusespa"></a>PROP_INET_USE_SPA

一般的なインターネット メールボックスのセキュリティで保護されたパスワード認証 (SPA) を使用するかどうかを指定します。
  
## <a name="quick-info"></a>クイック ヒント

|||
|:-----|:-----|
|識別子:  <br/> |0x0108  <br/> |
|プロパティの種類:  <br/> |PT_DWORD  <br/> |
|プロパティ タグ。  <br/> |0x01080003  <br/> |
|アクセス:  <br/> |読み取り専用  <br/> |
   
## <a name="remarks"></a>注釈

ゼロの値の SPA を使用して、SPA を使用して、それ以外の場合はありません。
