# <a name="outlook-weather-location-xml-schema"></a>Outlook Weather Location XML スキーマ

このセクションには、outlook の天気予報の XML スキーマに関する outlook 2013 スキーマリファレンスを記載しています。 このスキーマを使用すると、天気予報サービスプロバイダーは、場所コードの Outlook web サービス要求に応答できます。 場所コードは、ユーザーが天気予報バーで天気予報情報を要求している場所に対応している必要があります。
