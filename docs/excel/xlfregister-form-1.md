### <a name="e-data-type"></a><span data-ttu-id="868a8-291">E データ型</span><span class="sxs-lookup"><span data-stu-id="868a8-291">E Data Type</span></span>

<span data-ttu-id="868a8-p117">Excel は、E データ型を使用する DLL がスタック上の浮動小数点数へのポインターを渡すことを期待します。これは、コプロセッサ エミュレーター スタック上の数値が渡されることを期待する一部の言語 (Borland C++ など) で問題を起こす可能性があります。回避策としては、コプロセッサ スタック上の数値へのポインターを渡します。次の例は、Borland C++ から double が返される方法を示しています。</span><span class="sxs-lookup"><span data-stu-id="868a8-p117">Excel expects a DLL using the E data type to pass pointers to floating-point numbers on the stack. ```cpp
typedef double * lpDbl;
extern "C" lpDbl __stdcall AddDbl(double D1,

    lpDbl Result;
