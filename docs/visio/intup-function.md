# <a name="intup-function"></a>INTUP 関数

次の整数に切り上げます。
  
## <a name="syntax"></a>構文

INTUP (* **番号** *) 
  
### <a name="parameters"></a>Parameters

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必須  <br/> |**番号** <br/> |切り上げの対象となる数値を指定します。  <br/> |
   
## <a name="example-1"></a>例 1

INTUP(3.2)
  
4 を返します。
  
## <a name="example-2"></a>例 2

INTUP(-3.2)
  
-3 を返します。
  
## <a name="example-3"></a>例 3

INTUP(3)
  
3 を返します。
