# <a name="type-a-new-formula-for-a-cell"></a>セルに新しい数式を入力する

1. セルを選択します。
    
    セルと数式バーにセルの数式が表示されます。
    
2. 新しい数式を入力します。
    
    入力した数式がセルの元の数式と置き換わります。
    
    測定単位を指定せずに数値を入力した場合、セルの以前の値の単位がある場合は、その単位が使用されます。 または、セルに単位が必要な場合は、セルの既定の単位が使用されます。
    
3. 数式を適用するには、Enter キーを押します。
    
    数値または数式にエラーが含まれている場合は、メッセージが表示され、数式バーまたはセルでエラーが強調表示されます。 エラーを修正し、Enter キーを押して修正を適用します。
