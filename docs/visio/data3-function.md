# <a name="data3-function"></a>DATA3 関数

図形の [**図形の名前**] ダイアログ ボックスで、**データ 3**のボックスからテキストを返します ([**開発**] タブで、**図形の名前**] をクリック)、文字列として。 
  
## <a name="syntax"></a>構文

DATA3 ()
  
## <a name="remarks"></a>注釈

ボックスが空の場合、関数は空の文字列を返します。
