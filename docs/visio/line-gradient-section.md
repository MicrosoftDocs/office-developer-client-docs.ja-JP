# <a name="line-gradient-section"></a>[線のグラデーション] セクション

図形に適用される線のグラデーションの色の値を表示します。 
  
## <a name="remarks"></a>Remarks

各[グラデーション停止の行 ([線のグラデーション])](gradient-stop-row-line-gradient-section.md)セクションの行では、線のグラデーションの分岐点を表します。 このセクションを空にすることができます。 
  
グラデーションの設定では、セクションに格納された最初の 10 行のみが使用されます。10 行を超える [**Gradient Stop**] 行はすべて無視されます。 
  
[**図形の書式設定**] ウィンドウの [**線**] セクションでは、線グラデーションのプロパティを設定できます ([**ホーム**] タブの [**図形のスタイル**] グループで、[**線**] をクリックしてから [**線のオプション**] をクリック)。 
  
有効にしたり、 [LineGradientEnabled (グラデーション プロパティ] セクションで)](linegradientenabled-cell-gradient-properties-section.md)セルの値を変更することによって、図形のグラデーションの線の色を無効にできます。
