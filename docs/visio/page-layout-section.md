# <a name="page-layout-section"></a>[Page Layout] セクション

ページ上にあるすべての図形間の間隔やコネクタ間の間隔、およびすべてのコネクタに対する迂回方法など、図形とコネクタに関するページ設定を制御するセルを格納します。
  
## <a name="remarks"></a>注釈

特定の図形に対するページの既定値を無効にするには、**図形のレイアウト**] セクションでセルを使用します。 
  
**[ページ設定**] ダイアログ ボックスで、[**レイアウトと経路**] タブで、このセクションのセルの値を設定することも ([**デザイン**] タブをクリックして [**ページ設定**] ダイアログ ボックス起動ツール)。
