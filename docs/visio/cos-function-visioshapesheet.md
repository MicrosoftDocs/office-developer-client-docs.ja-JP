# <a name="cos-function-visioshapesheet"></a>COS 関数 (VisioShapeSheet)

角度のコサインを返します。 
  
## <a name="syntax"></a>構文

COS (*角度*) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _直交_ <br/> |必須  <br/> |**数値型 (Numeric)** <br/> |コサインを取得する角度を指定します。  <br/> |
   
## <a name="remarks"></a>解説

COS 関数では、*角度*に別の単位を指定しない限り、ラジアンが使用されます。 
  
## <a name="example"></a>例

COS(45 deg) 
  
0.7071 を返します。
