# <a name="scratch-section"></a>[Scratch] セクション

他のセルが参照する数式の入力およびテストを実行するための作業領域です。
  
## <a name="remarks"></a>注釈

**セクションの挿入**] ダイアログ ボックスを使用して、このセクションを追加できます。 [シェイプ シート] ウィンドウ内を右クリックし、**セクションの挿入**] をクリックします。
  
繰り返される複雑な計算を分離するのには、[**スクラッチ**] セクションが使用されます。 ソリューションに適切に定義された目的がある場合は、ユーザー セルに名前を指定できるのでわかりやすくするための **[User-Defined Cells** ] セクションでセルを使用するが賢明です。 
  
[**スクラッチ**] セクションのセルは、2 つの方法で単位を使用します。 X セルと Y セルを使用して、図面の寸法単位です。D のセルには、ユニットを使用しないでください。 (C プログラマの専門用語で X と Y のセルは、型を指定、および D からのセルは、"void")**[Pinx]** と **[piny]**、または**ジオメトリ**セクションでは X と Y のセルなど、 *x 座標* *y*座標を派生するため、**スクラッチ****スクラッチ**セルが使用されます。 A ~ D を指定する単位数を表示できるセルをスクラッチします。 
  
さらに大きな違いは、これらのセルがポイント値を格納する方法です。 Visio におけるポイントとは、( *x, y*) 座標に対する単一のデータ パッケージです。 数式がポイント値を返す数式がシェイプ シート セルに応じて、3 つの方法のいずれかでその値が解釈されます。 *X*に関連するセルの (たとえば、 **[pinx]**、または **[Geometry** ] セクションの X カラム内のセル) の座標が*x*だけを抽出・ ポイントの値の一部を調整します。 *Y*に関連するセルの座標、 *y*だけを抽出する-ポイントの値の一部を調整します。 
  
Visio で数式を抽出するなど、`PNT(3,4)`では、これら 3 つの方法。 
  
|**Cell**|**入力する場合**|**Visio として扱われます**|**結果**|
|:-----|:-----|:-----|:-----|
| X  <br/> | `PNT(3,4)` <br/> | `PNTX(PNT(3,4))` <br/> | 3.0000 に示されます。  <br/> |
| Y  <br/> | `PNT(3,4)` <br/> | `PNTY(PNT(3,4))` <br/> | 4.0000 に示されます。  <br/> |
| A ~ D  <br/> | `PNT(3,4)` <br/> | `PNT(3,4)` <br/> | Pnt 関数 (3.0000 インチ、4.0000 で)。  <br/> |
