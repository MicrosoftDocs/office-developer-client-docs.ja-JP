# <a name="stencilgroup-element-windowtype-complextype-visio-xml"></a>StencilGroup 要素 (Window_Type complexType)'Visio XML (')

差し込み印刷のステンシル ウィンドウのうち、ウィンドウは、メンバーのグループを指定します。
  
## <a name="element-information"></a>要素情報

|||
|:-----|:-----|
|**要素の種類** <br/> |[StencilGroup_Type](stencilgroup_type-complextypevisio-xml.md) <br/> |
|**名前空間** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**スキーマ ファイル** <br/> |VisioSchema15.xsd  <br/> |
|**文書パーツ** <br/> |windows.xml  <br/> |
   
## <a name="definition"></a>定義

```XML
< xs:element name="StencilGroup" type="StencilGroup_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```

## <a name="elements-and-attributes"></a>要素と属性

スキーマは、**シーケンス**、 **minOccurs**、 **maxOccurs**では、**選択**などの特定の要件を定義する場合は、定義のセクションを参照してください。

  <br/> |
