# <a name="pi-function"></a>PI 関数

約 3.1415926535898 に数学定数*π*を返します。 
  
## <a name="syntax"></a>構文

PI( )
  
### <a name="return-value"></a>戻り値

数値
  
## <a name="example"></a>例

(14 インチ/2) ^2 \* PI) 
  
153.9380 インチ (直径 14 インチの円の面積) を返します。
