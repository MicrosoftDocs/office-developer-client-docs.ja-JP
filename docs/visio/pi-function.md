# <a name="pi-function"></a>PI 関数

数学的な定数*pi*を返します。これは、およそ3.1415926535898 です。 
  
## <a name="syntax"></a>構文

PI( )
  
### <a name="return-value"></a>戻り値

数値
  
## <a name="example"></a>例

(14 in/2) ^ 2 \* PI () 
  
153.9380 インチ (直径 14 インチの円の面積) を返します。
