# <a name="protection-section"></a>[保護] セクション

ロックの設定を現在の設定 ([[開発](run-in-developer-mode-display-the-developer-tab.md)] タブ) の [**保護**] コマンドと [シェイプ シート] ウィンドウでのみ設定できるいくつかの追加のロックを示しています。 
  
## <a name="remarks"></a>注釈

ロック機能によって、不注意による図形の変更を防ぐことができますが、他の状況では値のリセットを防ぐことができません。また [シェイプシート] ウィンドウでの変更から保護することもできません。
  
変更されている数式を保護するには、 **GUARD**関数を使用します。
