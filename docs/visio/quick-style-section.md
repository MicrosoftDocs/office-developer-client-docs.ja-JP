# <a name="quick-style-section"></a>[クイック スタイル] セクション

線の色とスタイル、塗りつぶしの色とスタイル、影の色、効果、フォント、およびスタイルの種類など、図形に適用されている現在のクイックスタイルの書式設定値を表示します。 
  
## <a name="remarks"></a>注釈

クイックスタイルを選択してから、新しいクイックスタイル ([**ホーム**] タブの [**図形のスタイル**] グループで [**その**他] をクリックしてから、ギャラリーから別のクイックスタイルを選択する) を選択することができます。
