# <a name="page-properties-section"></a>[Page Properties] セクション

ページの幅、高さ、縮尺などの、ページの属性を制御するセルを格納します。
  
## <a name="remarks"></a>注釈

**ページのプロパティ**] セクションのセルに入力をすることによって、または、[**ページ設定**] ダイアログ ボックスを使用してページのプロパティの値を設定することができます ([**デザイン**] タブをクリックして [**ページ設定**] ダイアログ ボックス起動ツール)。
