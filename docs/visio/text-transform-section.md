## <a name="remarks"></a>注釈

このセクションを追加するには、**セクションの挿入**] ダイアログ ボックスを使用して (シェイプ シート ウィンドウ内を右クリックし、[**セクションの挿入**] をクリック) します。 **Text Transform** ] セクションに値を設定することは、**テキスト ブロック**] ツール ([**ホーム**] タブの [**ツール**] グループ) を使用してテキスト ブロックを処理するのには相当します。
