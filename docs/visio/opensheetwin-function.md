# <a name="opensheetwin-function"></a>OPENSHEETWIN 関数

新しいウィンドウで [シェイプシート] ウィンドウを開きます。
  
## <a name="syntax"></a>構文

OPENSHEETWIN( )
  
## <a name="remarks"></a>解説

以前のバージョンの Visio では、この関数は _OPENSHEETWIN に相当します。Visio 4.0 以降のバージョンでは、どちらのスタイルも使用できます。
