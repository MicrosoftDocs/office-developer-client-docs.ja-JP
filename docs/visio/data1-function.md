# <a name="data1-function"></a>DATA1 関数

図形の [**図形の名前**] ダイアログ ボックスで [**メモ 1** ] ボックスからテキストを返します ([**開発**] タブで、**図形の名前**] をクリック)、文字列として。 
  
## <a name="syntax"></a>構文

DATA1 ()
  
## <a name="remarks"></a>注釈

ボックスが空の場合、関数は空の文字列を返します。
