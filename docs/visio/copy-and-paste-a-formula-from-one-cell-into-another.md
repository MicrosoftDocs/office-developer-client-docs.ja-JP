# <a name="copy-and-paste-a-formula-from-one-cell-into-another"></a>セル間での数式のコピーおよび貼り付け

1. コピーする数式を含むセルを選択します。
    
2. CTRL キーを押しながら C キーを押します。
    
    数式全体を置き換えるには、置換する数式を含むセルを選択します。
    
    既存の数式に、数式を貼り付ける、セルをダブルクリックし、数式を貼り付ける挿入点をクリックし、します。
    
3. Ctrl キーを押しながら V キーを押します。
    
4. 数式を適用する場合は、Enter キーを押します。
