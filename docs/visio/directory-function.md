# <a name="directory-function"></a>DIRECTORY 関数

ドキュメントが保存されている、文字列としてのディレクトリ (フォルダー) への完全パスを返します。 ドキュメントが保存されていない場合は、空の文字列を返します。
  
## <a name="syntax"></a>構文

DIRECTORY ()
