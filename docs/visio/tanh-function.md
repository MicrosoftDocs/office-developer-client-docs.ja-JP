# <a name="tanh-function"></a>TANH 関数

角度の双曲正接を返します。 
  
## <a name="syntax"></a>構文

TANH (* * *angle* * *) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _直交_ <br/> |必須  <br/> |**数値** <br/> |双曲タンジェントを取得する角度を指定します。  <br/> |
   
### <a name="return-value"></a>戻り値

数値
  
## <a name="remarks"></a>注釈

TANH 関数では、*角度*に別の単位を指定しない限り、ラジアンが使用されます。 
  
## <a name="example"></a>例

TANH (-45) 
  
-0.6558 を返します。
