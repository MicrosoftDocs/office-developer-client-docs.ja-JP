# <a name="text-block-format-section"></a>[Text Block Format] セクション

図形のテキスト ブロックでのテキストの配置、余白、およびタブ位置を制御するセルを格納します。
  
## <a name="remarks"></a>解説

[**テキスト**] ダイアログボックスを使用してこれらのテキスト属性を設定することもできます ([**ホーム**] タブの [**フォント**] グループで、ダイアログボックス起動ツールをクリックします)。
