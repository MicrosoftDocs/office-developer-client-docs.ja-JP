# <a name="fonttoid-function"></a>FONTTOID 関数

指定されたフォントの識別子 (ID) を返します。
  
> [!NOTE]
> **FONTTOID**関数に似た[FONT](font-function.md)関数を使用することをお勧めします。 Microsoft Visio 2013 では、 **FONTTOID**を置換するために**FONT**関数が導入されています。 
  
## <a name="syntax"></a>構文

FONTTOID (* * *font_name* * *) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _font_name_ <br/> |必須  <br/> |**String** <br/> |フォントの名前。  <br/> |
