# <a name="fonttoid-function"></a>FONTTOID 関数

指定されたフォントの識別子 (ID) を返します。
  
> [!NOTE]
> **FONTTOID**関数のような[フォント](font-function.md)関数を使用することをお勧めします。 Microsoft Visio 2013 では、 **FONTTOID**を置換する**フォント**機能が登場しました。 
  
## <a name="syntax"></a>構文

FONTTOID (* * *font_name* * *) 
  
### <a name="parameters"></a>Parameters

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _font_name_ <br/> |必須  <br/> |**String** <br/> |フォント名を指定します。  <br/> |
