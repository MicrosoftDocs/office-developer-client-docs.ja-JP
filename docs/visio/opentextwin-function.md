# <a name="opentextwin-function"></a>OPENTEXTWIN 関数

図形のテキストブロックを開いて、テキストを編集できるようにします。
  
## <a name="syntax"></a>構文

OPENTEXTWIN( )
  
## <a name="remarks"></a>解説

以前のバージョンの Visio では、この関数は _OPENTEXTWIN に相当します。Visio 4.0 以降のバージョンでは、どちらのスタイルも使用できます。
