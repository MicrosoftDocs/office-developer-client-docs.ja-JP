# <a name="layer-membership-section"></a>[Layer Membership] セクション

図形が割り当てられる各レイヤーを一覧表示する行を格納します。
  
## <a name="remarks"></a>注釈

レイヤーの割り当ては、ページ上のレイヤーの一覧へのインデックスとして表示されます。 レイヤーのインデックスが、[**レイヤー プロパティ**] ダイアログ ボックスでレイヤーの順序に対応する ([**ホーム**] タブの [**編集**]、[**レイヤー**] をクリックし、[**レイヤー プロパティ**] をクリック) します。 ダイアログ ボックスの最初の名前は、レイヤー 0、2 番目はレイヤー 1 などです。
  
図形が複数のレイヤーに割り当てられている場合、各レイヤーのインデックスは、[Layer Membership] セルに、セミコロンで区切られて表示されます。
  
数式で [Layer Membership] セルの値を参照するには、 **LayerMember**の名前を使用します。
