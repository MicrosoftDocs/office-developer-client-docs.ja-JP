# <a name="doclastedit-function"></a>DOCLASTEDIT 関数

図面が最後に編集された日時をシリアル値として返します。
  
## <a name="syntax"></a>構文

DOCLASTEDIT()
  
## <a name="remarks"></a>注釈

値の書式を設定するのにには、 [FORMAT](format-function.md)関数を使用します。
