# <a name="character-section"></a>[Character] セクション

フォント、色、テキストのスタイル、大文字/小文字、上下位置の調整、およびポイント サイズなど、図形のテキストについての書式設定属性を表示します。
  
## <a name="remarks"></a>解説

図形の各文字スタイルには、[シェイプシート] ウィンドウに表示される [**文字**] セクションの対応する行があります。 行の名前は、その文字スタイルが適用される文字の番号を表します。 
  
テキストの書式を設定するには、[**テキスト**] ダイアログボックス ([**ホーム**] タブの [**フォント**] グループでダイアログボックス起動ツールをクリック)、テキストスタイルを適用するか、[ **Character** ] セクションのセルでエントリを作成します。
