# <a name="tan-function-visioshapesheet"></a>TAN 関数 (VisioShapeSheet)

角度のタンジェントを返します。
  
## <a name="syntax"></a>構文

TAN (* **角度** *) 
  
### <a name="parameters"></a>Parameters

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _角度_ <br/> |必須  <br/> |**数値** <br/> |タンジェントを取得する対象の角度です。  <br/> |
   
### <a name="return-value"></a>�߂�l

数値型 (Numeric)
  
## <a name="remarks"></a>注釈

TAN 関数では、*角度*で別の測定単位を指定する場合を除き、ラジアンが使用されます。 
  
## <a name="example"></a>例

TAN(45 deg) 
  
1 を返します。
