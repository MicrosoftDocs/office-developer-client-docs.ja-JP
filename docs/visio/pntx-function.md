# <a name="pntx-function"></a>PNTX 関数

点の_x_座標を返します。
  
## <a name="syntax"></a>構文

PNTX (* * *point* * *) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _ポイント_ <br/> |必須  <br/> |**数値** <br/> |点の_x_座標を指定します。  <br/> |
   
### <a name="return-value"></a>戻り値

番号
  
## <a name="example"></a>例

PNTX (PNT (7, 12)) 
  
7 を返します。
