# <a name="pntx-function"></a>PNTX 関数

ポイントの  _x_ 座標を返します。
  
## <a name="syntax"></a>構文

PNTX(** *point* ** ) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _point_ <br/> |必須  <br/> |**数値** <br/> |ポイント  _の x_ 座標。  <br/> |
   
### <a name="return-value"></a>戻り値

番号
  
## <a name="example"></a>例

PNTX(PNT(7,12)) 
  
7 を返します。
