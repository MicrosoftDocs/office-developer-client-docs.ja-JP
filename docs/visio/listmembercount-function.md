# <a name="listmembercount-function"></a>LISTMEMBERCOUNT 関数

リスト コンテナー図形内のメンバー図形の数を返します。
  
## <a name="version-information"></a>バージョン情報

追加バージョン: Visio 2010
 
  
## <a name="syntax"></a>構文

LISTMEMBERCOUNT()
  
### <a name="return-value"></a>戻り値

整数
  
## <a name="remarks"></a>注釈

図形がリスト コンテナーではない場合、LISTMEMBERCOUNT 関数は 0 を返します。
