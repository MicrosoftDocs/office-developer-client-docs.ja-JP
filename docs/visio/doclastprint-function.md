# <a name="doclastprint-function"></a>DOCLASTPRINT 関数

ドキュメントが最後に印刷されたシリアル値として日付を返します。
  
## <a name="syntax"></a>構文

DOCLASTPRINT ()
  
## <a name="remarks"></a>注釈

戻り値を書式設定するには、FORMAT 関数を使用します。
