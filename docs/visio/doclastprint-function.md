# <a name="doclastprint-function"></a>DOCLASTPRINT 関数

ドキュメントが最後に印刷された日付と時刻をシリアル値として返します。
  
## <a name="syntax"></a>構文

DOCLASTPRINT ()
  
## <a name="remarks"></a>解説

戻り値を書式設定するには、FORMAT 関数を使用します。
