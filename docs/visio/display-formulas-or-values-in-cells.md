# <a name="display-formulas-or-values-in-cells"></a>セルの数式または値を表示する

1. [シェイプシート] ウィンドウをクリックしてアクティブにします。
    
2. **シェイプ ツール**] タブの [**ビュー** ] グループで、数式を表示するのには**数式**をクリックするか、値を表示する**値**] をクリックします。 
    
    > [!NOTE]
    > シェイプ シート ウィンドウで右クリックしてもし、ショートカット メニューの [**数式**または**値**をクリックできます。
