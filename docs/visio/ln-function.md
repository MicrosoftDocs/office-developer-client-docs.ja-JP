# <a name="ln-function"></a>LN 関数

数値の自然対数を返します。 数は正の値である必要があります。 または行が、エラー値 #num を返します。
  
## <a name="syntax"></a>構文

LN (* **番号** *) 
  
### <a name="parameters"></a>Parameters

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必須  <br/> |**数値** <br/> | 自然対数を調べる数値を指定します。  <br/> |
