# <a name="footerleft-element-headerfootertype-complextype-visio-xml"></a>フッター left 要素 (HeaderFooter_Type complexType) (' Visio XML ')

## <a name="element-information"></a>要素情報

|||
|:-----|:-----|
|**要素の種類** <br/> |[FooterLeft_Type](footerleft_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**スキーマ ファイル** <br/> |VisioSchema15  <br/> |
|**文書パーツ** <br/> |文書の xml  <br/> |
   
## <a name="definition"></a>定義

```XML
< xs:element name="FooterLeft" type="FooterLeft_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```

スキーマで**sequence**、 **minOccurs**、 **maxOccurs**、 **choice**などの特定の要件が定義されている場合は、「定義」セクションを参照してください。
