# <a name="gradient-properties-section"></a>[グラデーションのプロパティ] セクション

図形に適用されているグラデーションの塗りつぶしまたは線の色の効果のプロパティを表示します。 このセクションには、図形のグラデーションを図形と一緒に回転するかどうか、または他の図形とグループ化したときに適用されるグラデーションを使用するかどうかを制御するセルも含まれます。 
  
## <a name="remarks"></a>解説

**** > **** > **** > [**図形の書式**設定] ダイアログボックスでグラデーションの設定を行うことができます ([**ホーム** > **図形のスタイル** > ] 塗りつぶしの**グラデーションの塗り**つぶしの塗りつぶしの色)。
