# <a name="sheetref-function"></a>SHEETREF 関数

_Sheetref_で指定されているシート (シェイプ) への参照を取得または、現在のシートに、 _sheetref_の修飾子がない場合。 
  
## <a name="version-information"></a>バージョン情報

追加バージョン: Visio 2010
 
  
## <a name="syntax"></a>構文

 *\<sheetref\>!*  SHEETREF() 
  
### <a name="return-value"></a>戻り値

シェイプシート参照
  
## <a name="remarks"></a>注釈

シート参照トークンを使用する他の関数でもこの関数を使用できます。
