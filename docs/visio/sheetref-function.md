# <a name="sheetref-function"></a>SHEETREF 関数

_sheetref_ で指定されているシート (図形) への参照、またはシート参照修飾子がない場合は現在のシートへの参照を返します。 
  
## <a name="version-information"></a>バージョン情報

追加バージョン: Visio 2010
 
  
## <a name="syntax"></a>構文

 *\<sheetref \> !*  SHEETREF() 
  
### <a name="return-value"></a>戻り値

シェイプシート参照
  
## <a name="remarks"></a>注釈

シート参照トークンを使用する他の関数でもこの関数を使用できます。
