# <a name="edit-a-part-of-a-formula"></a>数式の一部を編集する

1. セルをダブルクリックします。または、セルを選択して F2 キーを押します。
    
2. セルまたは数式バーで、次の表に示す方法に従って数式を編集します。
    
|**実行内容**|**行動**|
|:-----|:-----|
| カーソルを置きます  <br/> | カーソルを表示する位置をクリックします。  <br/> |
| カーソルを移動します。  <br/> | 左または右矢印キーを押します。  <br/> |
| 左側の文字を削除します。  <br/> | BACKSPACE キーを押します。  <br/> |
| 右側の文字を削除します。  <br/> | Del キーを押します。  <br/> |
| テキストを選択します。  <br/> | テキスト上でポインターをドラッグします。  <br/> |
| 左へ選択範囲を拡張します。  <br/> | Shift キーを押しながら左矢印キーを押します。  <br/> |
| 右側に選択範囲を拡張します。  <br/> | Shift キーを押しながら右矢印キーを押します。  <br/> |
| 単語を選択します。  <br/> | CTRL + シフト + 右矢印を押すか、ダブルクリックします。  <br/> |
| 末尾まで選択します。  <br/> | Shift キーを押しながら END キーを押します。  <br/> |
| 先頭まで選択します。  <br/> | Shift キーを押しながら HOME キーを押します。  <br/> |
| 選択したテキストを置き換える  <br/> | 新しいテキストを入力します。  <br/> |
| 選択したテキストを削除します。  <br/> | Del キーを押します。 または、[**ホーム**] タブの [**切り取り**] をクリックします。  <br/> |
