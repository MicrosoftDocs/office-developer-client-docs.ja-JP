# <a name="deg-function"></a>DEG 関数

角度の値をラジアンから度に変換します。
  
## <a name="syntax"></a>構文

DEG (* **角度** *) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _角度_ <br/> |必須  <br/> |**文字列型 (String)** <br/> |角度の値をラジアンで指定します。  <br/> |
   
## <a name="example"></a>例

DEG(PI( )/4) 
  
45 を返します
