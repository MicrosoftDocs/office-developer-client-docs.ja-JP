# <a name="is1d-function"></a>IS1D 関数

図形が 1-D (1 次元) の場合、TRUE を返します。図形が 2-D (2 次元) の場合、FALSE を返します。
  
## <a name="version-information"></a>バージョン情報

Visio 2010 のバージョンが追加されます。 
  
## <a name="syntax"></a>構文
