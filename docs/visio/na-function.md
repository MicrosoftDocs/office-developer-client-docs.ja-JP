# <a name="na-function"></a>NA 関数

エラー値 #NA! を返します。 
  
情報が含まれていないシェイプシートのセルで NA 関数を使用すると、セルの既定値が計算で使用されなくなります。
  
## <a name="syntax"></a>構文

NA( )
  
### <a name="return-value"></a>戻り値

文字列
  
## <a name="remarks"></a>注釈

NA は "使用可能な値がない" ことを意味するエラー値です。
