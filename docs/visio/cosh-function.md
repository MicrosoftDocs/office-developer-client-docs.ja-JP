# <a name="cosh-function"></a>COSH 関数

角度の双曲線余弦を返します。
  
## <a name="syntax"></a>構文

COSH (*角度*) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _角度_ <br/> |必須  <br/> |**数値型 (Numeric)** <br/> |双曲線余弦 (ハイパーボリック コサイン) を取得する角度を指定します。  <br/> |
   
## <a name="remarks"></a>注釈

COSH 関数では、*角度*で別の測定単位を指定する場合を除き、ラジアンが使用されます。 
  
## <a name="example"></a>例

COSH(45 deg) 
  
1.3246 を返します。
