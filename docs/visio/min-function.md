# <a name="min-function"></a>MIN 関数

リストから最小値を返します。 負の無限大に最も近い最小の手段です。
  
## <a name="syntax"></a>構文

最小 (* * *[数値 1]* * *、* **数値 2* * *,...、* * *numberN* * *) 
  
### <a name="parameters"></a>Parameters

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _number1_ <br/> |必須  <br/> |**によって異なります** <br/> |リスト内の最初の数値を指定します。  <br/> |
| _number2_ <br/> |省略可能  <br/> |**によって異なります** <br/> | リスト内の 2 番目の数値を指定します。  <br/> |
| _numberN_ <br/> |省略可能  <br/> |**によって異なります** <br/> |リスト内の n 番目の数値を指定します。  <br/> |

可変値

MIN(13 in,1 ft, 20 cm)
