# <a name="min-function"></a>MIN 関数

リストから最小の数を返します。 最小は負の無限大に最も近い意味です。
  
## <a name="syntax"></a>構文

MIN(** *number1* **, ** *number2* **,..., ** *numberN* ** ) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _number1_ <br/> |必須  <br/> |**さまざま** <br/> |リスト内の最初の数値を指定します。  <br/> |
| _number2_ <br/> |省略可能  <br/> |**さまざま** <br/> | リスト内の 2 番目の数値を指定します。  <br/> |
| _numberN_ <br/> |省略可能  <br/> |**さまざま** <br/> |リスト内の n 番目の数値を指定します。  <br/> |
   
### <a name="return-value"></a>戻り値

さまざま
  
## <a name="example"></a>例

MIN(13 in,1 ft, 20 cm) 
  
20 cm を返します。
