# <a name="acos-function"></a>ACOS 関数

数値のアークコサイン (たとえば、コサインが数値である角度) を返  *します*  。 
  
## <a name="syntax"></a>構文

ACOS( *number*) 
  
## <a name="remarks"></a>注釈

入力値は、-1 = number = 1 の範囲にある \< \< 必要#NUM。 エラーを返します。 結果の角度は、範囲 0 = angle = PI ラジアン \< \< (0 \< = angle = \< 180 度) です。
  
## <a name="example"></a>例

ACOS(0) 
  
90°を返します。
