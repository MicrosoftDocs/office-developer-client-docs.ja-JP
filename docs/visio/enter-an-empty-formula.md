# <a name="enter-an-empty-formula"></a>空の数式を入力する

1. 数式を含めるセルを選択します。
    
2. Del キーを押してから、Enter キーを押します。
    
> [!NOTE]
> **スクラッチ**セルのローカルの数式を挿入した後これをください。 セルは、マスター シェイプから継承する数式がないために、' いいえ数式 ' に戻ります。
