# <a name="enter-an-empty-formula"></a>空の数式を入力する

1. 数式を含めるセルを選択します。
    
2. Del キーを押してから、Enter キーを押します。
    
> [!NOTE]
> ローカルの数式を挿入した後、それを**スクラッチ**のセルで試してみてください。 マスターから継承する数式がないので、セルは [数式なし] に戻ります。
