# <a name="doccreation-function"></a>DOCCREATION 関数

シリアル値として、ドキュメントが作成された日時を返します。
  
## <a name="syntax"></a>構文

DOCCREATION ()
  
## <a name="remarks"></a>注釈

戻り値を書式設定するには、FORMAT 関数を使用します。
