# <a name="abs-function-visioshapesheet"></a>ABS 関数 (VisioShapeSheet)

数値の絶対値を返します。
  
## <a name="syntax"></a>構文

ABS (*数値*) 
  
### <a name="parameters"></a>パラメーター

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _Number_ <br/> |必須  <br/> |**Decimal** <br/> |絶対値を調べる数値を指定します。  <br/> |
   
## <a name="example"></a>例

ABS(-1.24) 
  
1.24 を返します。
