# <a name="type-function"></a>TYPE 関数

オブジェクトの内部型を数値として返します。 
  
## <a name="syntax"></a>構文

TYPE ()
  
### <a name="return-value"></a>戻り値

番号

|グループ  <br/> |71  <br/> |
|Shape  <br/> |72  <br/> |
|Style  <br/> |74  <br/> |
|ドキュメント  <br/> |77  <br/> |
|Foreign  <br/> |78  <br/> |
|Document  <br/> |79  <br/> |
