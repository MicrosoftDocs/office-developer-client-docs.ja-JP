# <a name="atan-function"></a>ATAN 関数

そのタンジェントが_数値_である角度など、数値のアーク タンジェントを返します。 結果として得られる角度は -π/2 の範囲で、< = 角度 < = π/2 ラジアン (-90 < = 角度 < = 90 °)。 
  
## <a name="syntax"></a>構文

ATAN (* **番号** *) 
  
### <a name="parameters"></a>Parameters

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必須  <br/> |**数値** <br/> |角度のタンジェントを指定します。  <br/> |
   
## <a name="example"></a>例

ATAN(1) 
  
45°を返します
