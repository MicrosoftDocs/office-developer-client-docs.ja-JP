# <a name="sin-function-visioshapesheet"></a>SIN 関数 (VisioShapeSheet)

角度の正弦を返します。 
  
## <a name="syntax"></a>構文

SIN (* **角度** *) 
  
### <a name="parameters"></a>Parameters

|**名前**|**必須 / オプション**|**データ型**|**説明**|
|:-----|:-----|:-----|:-----|
| _角度_ <br/> |必須  <br/> |**数値** <br/> |サインを取得する角度を指定します。  <br/> |

数値
  
## <a name="remarks"></a>備考

SIN 関数では、_角度_で別の測定単位を指定する場合を除き、ラジアンが使用されます。

0.7071 を返します。
