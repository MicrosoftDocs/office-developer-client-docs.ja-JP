# <a name="shape-layout-section"></a>[Shape Layout] セクション

図形の配置およびコネクタの迂回方法を制御するセルを格納します。
  
## <a name="remarks"></a>注釈

**動作**] ダイアログ ボックスで、[**配置**] タブを使用して、このセクションの図形固有のセルの値を設定することもできます ([[開発](run-in-developer-mode-display-the-developer-tab.md)] タブの [**図形のデザイン**] で、[**動作**] をクリック) します。 **動作**] ダイアログ ボックスで、[**コネクタ**] タブを使用してこのセクションでは、コネクタ固有のセルの値を設定できます。 ( **[コネクタ**] タブを表示するには、図面ウィンドウでコネクタを選択、[**動作**] ダイアログ ボックスを表示) 
  
**ページ レイアウト**] セクションに含まれる既定値をオーバーライドするのには [**図形のレイアウト**] セクションの値を使用できます。
