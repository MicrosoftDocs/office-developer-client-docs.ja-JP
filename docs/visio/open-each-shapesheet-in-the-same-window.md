# <a name="open-each-shapesheet-in-the-same-window"></a>各シェイプシートを同じウィンドウ内で開く

1. [**ファイル**] タブをクリックし、[**オプション**] をクリックして、[**詳細設定**] をクリックします。
    
2. [**全般**] の [**各シェイプシートを同じウィンドウ内で開く**] を選択します。
    
3. **[OK]** をクリックします。
