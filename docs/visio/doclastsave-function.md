# <a name="doclastsave-function"></a>DOCLASTSAVE 関数

ドキュメントが最後に保存された、シリアル値として日付を返します。
  
## <a name="syntax"></a>構文

DOCLASTSAVE ()
  
## <a name="remarks"></a>注釈

戻り値を書式設定するには、FORMAT 関数を使用します。
