# <a name="project-programming-tasks"></a>Project のプログラミング タスク

このセクションには、JavaScript ライブラリをクライアント側オブジェクトモデル (csom) で使用する方法を示すいくつかの操作方法に関する記事と、project Server 2013 および project Online の他のプログラミングタスクを実行する方法が記載されています。 プログラミングタスクの例としては、SharePoint でホストされる Project Server アプリの作成、需要管理のためのワークフローの作成などがあります。Project Server アプリケーションを Windows Communication Foundation (WCF) を使用してプログラミングする。Project Web App のリボンをカスタマイズするProject Server Web パーツの作成。Project Server イベントハンドラとリモートイベントレシーバーを作成するまたは、ユーザー設定フィールドを一括更新し、project Online 用のプロジェクトサイトを作成します。
  
> [!NOTE]
> js grid コントロールを使用したプログラミングの詳細については、「SharePoint 2010 開発者用リファレンス」の「 [js grid control](https://msdn.microsoft.com/library/ee535898%28office.14%29.aspx) 」を参照してください。 マネージ コード リファレンスについては、「[Microsoft.SharePoint.JSGrid Namespace](https://msdn.microsoft.com/library/microsoft.sharepoint.jsgrid%28Office.15%29.aspx)」を参照してください。 JS Grid コントロール web コントロールの場合は、「 [JSGrid class](https://msdn.microsoft.com/library/microsoft.sharepoint.webcontrols.jsgrid%28Office.15%29.aspx)」を参照してください。
