# <a name="log-function-access-custom-web-app"></a>ログ機能 (カスタム web アプリケーションのアクセス)

指定した式の自然対数または指定した値を底とする対数を返します。
  
> [!IMPORTANT]
> [!重要] マイクロソフトを作成して、sharepoint web アプリケーションのアクセスを使用して不要になったをお勧めします。代わりに、web およびモバイル デバイス用のコードのないビジネス ソリューションを構築する[マイクロソフトの PowerApps](https://powerapps.microsoft.com/en-us/)を使用して検討してください。 
  
## <a name="syntax"></a>構文

 **ログ**(*数式*を [*基本*]) 
  
**Log** 関数には次の引数があります。 
  
|**引数名**|**説明**|
|:-----|:-----|
| *数式*  <br/> |対数が返される正の値。  <br/> |
| *Base*  <br/> |対数の底。 省略すると、**ログ**機能は、自然対数を返します。  <br/> |
   
## <a name="remarks"></a>解説

類似する **Log10** 関数では、常に底が 10 である常用対数が返されます。 
  
自然対数は e を底とする対数 2.718281828 にほぼ等しく、非合理的な定数です。
