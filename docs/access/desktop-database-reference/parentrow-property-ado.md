>>>>>>> master


**適用されます**Access 2013 |。Office 2013


OLE DB **Row** オブジェクトのコンテナーを **ADORecordConstruction** オブジェクトに設定し、行の親が ADO **Record** オブジェクトに変換されるようにします。

値の設定のみ可能です。

## <a name="syntax"></a>構文

HRESULT に\_ParentRow (\[の\]IUnknown\* pParent)。

## <a name="parameters"></a>パラメーター

  - *pParent*

  - 行のコンテナー。
