# <a name="tabledefs-members-dao"></a>テーブル定義のメンバー (DAO)


**適用されます**Access 2013、Office 2013。

TableDefs コレクションには、データベースに格納されているすべての TableDef オブジェクトが含まれます (Microsoft Access ワークスペースのみ)。
