# <a name="databaserelations-property-dao"></a>Relations プロパティ (DAO)


**適用先:** Access 2013、Office 2013

指定したデータベースの、すべての保存された **Relation** オブジェクトを含む **Relations** コレクションを返します。 読み取り専用です。

## <a name="syntax"></a>構文

*式*。Relations

*式***Database**オブジェクトを表す変数を取得します。
