# <a name="exitforeachrecord-macro-action"></a>"ExitForEachRecord/レコードごとに終了" マクロ アクション


**適用されます**Access 2013 |。Office 2013

" **ExitForEachRecord/レコードごとに終了** " アクションを使用して、 **[ForEachRecord](foreachrecord-data-block.md)** データ ブロックを即座に終了できます。


> [!NOTE]
> <P>[!メモ] " <STRONG>ExitForEachRecord/レコードごとに終了</STRONG> " アクションは、データ マクロでのみ使用できます。</P>
