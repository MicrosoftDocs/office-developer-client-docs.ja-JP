# <a name="comment-macro-statement"></a>Comment マクロ ステートメント


**適用されます**Access 2013、Office 2013。

**Comment** ステートメントを使用して、マクロに説明コメントを指定できます。

## <a name="remarks"></a>備考

**Comment** には、最大 1000 文字まで使用できます。ただし、既定ではコメントの先頭行のみが表示されます。
