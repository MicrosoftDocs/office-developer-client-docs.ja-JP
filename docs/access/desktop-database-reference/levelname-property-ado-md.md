# <a name="levelname-property-ado-md"></a>LevelName プロパティ (ADO MD)


**適用されます**Access 2013 |。Office 2013

メンバーのレベル名を示します。

## <a name="return-values"></a>戻り値

文字列型 ( **String** ) の値を取得します。値の取得のみが可能です。

## <a name="remarks"></a>解説

**LevelName** プロパティを使用して、メンバーが属しているレベルの名前を取得します。このプロパティは、 [Level](name-property-ado-md.md) オブジェクトの [Name](level-object-ado-md.md) プロパティに対応します。
