# <a name="view-an-activex-control-about-box"></a>ActiveX コントロールのバージョン情報の表示

**適用先:** Access 2013、Office 2013

ActiveX コントロールの**** バージョンと著作権情報を示す情報ボックスを表示するには、次の操作を行います。

1. access のプロパティシートで [**バージョン情報**] ボックスを選択します。

2. プロパティボックスの右側にある [**ビルド**] をクリックします。
