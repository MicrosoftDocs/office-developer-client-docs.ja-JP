# <a name="recordsetdatecreated-property-dao"></a>Recordset.DateCreated プロパティ (DAO)


**適用されます**Access 2013、Office 2013。

ベース テーブルを作成した日付と時刻を取得します (Microsoft Access ワークスペースのみ)。値の取得のみ可能です。バリアント型 ( **Variant**) の値を使用します。

## <a name="syntax"></a>構文

*式*です。作成日時

*式***レコード セット**オブジェクトを表す変数です。

## <a name="remarks"></a>注釈

日付と時刻の設定は、ベース テーブルを作成した、または最後に更新したコンピューターから導出されます。
