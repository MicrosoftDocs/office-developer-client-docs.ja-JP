<span data-ttu-id="d74a4-p104">Java Type Library ウィザードは、所定のタイプ ライブラリにある列挙型を INT (整数) 型に変換します。また、タイプ ライブラリの各列挙型に対応するインターフェイスも定義します。ADO 列挙型の値は、次の構文で参照できます。</span><span class="sxs-lookup"><span data-stu-id="d74a4-p104">The Java Type Library Wizard converts enumerated types, found in any given type library, to type INT (integer). ```vb 
 
msado15.<Enum Name>.<constant Name> 
```

<span data-ttu-id="d74a4-114">次に示す **Command** オブジェクトの **CommandType** プロパティを設定するコードは、この構文の例を示したものです。</span><span class="sxs-lookup"><span data-stu-id="d74a4-114">An example of this is shown in the following code fragment for setting the **CommandType** property of a **Command** object:</span></span>

```
