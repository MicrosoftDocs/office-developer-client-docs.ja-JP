# <a name="field2complextype-property-dao"></a>Field2 プロパティ (DAO)


**適用先:** Access 2013、Office 2013

複数値を持つフィールドを表す **[ComplexType](complextype-object-dao.md)** オブジェクトを返します。 値の取得および設定が可能です。

追加されたバージョン: Access 2007
