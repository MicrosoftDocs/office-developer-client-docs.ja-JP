# <a name="documentdatecreated-property-dao"></a>datecreated プロパティプロパティ (DAO)


**適用先:** Access 2013、Office 2013

オブジェクトが作成された日付と時刻を返します。 バリアント型 (**Variant**) の値を使用します。

## <a name="syntax"></a>構文

*式*。datecreated プロパティ

*式***Document**オブジェクトを表す変数を取得します。
