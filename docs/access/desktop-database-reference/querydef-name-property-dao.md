# <a name="querydefname-property-dao"></a>QueryDef.Name プロパティ (DAO)


**適用されます**Access 2013、Office 2013。

指定したオブジェクトの名前を取得または設定します。値の取得および設定が可能です。文字列型 ( **String**) の値を使用します。

## <a name="syntax"></a>構文

*式*です。名

*式***クエリ定義**オブジェクトを表す変数です。

## <a name="remarks"></a>注釈
