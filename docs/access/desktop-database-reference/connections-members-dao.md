# <a name="connections-members-dao"></a>Connections メンバー (DAO)

**適用先:** Access 2013、Office 2013

> [!NOTE]
> [!メモ] Microsoft Access 2013 では、ODBCDirect ワークスペースはサポートされていません。 Microsoft Access データベース エンジンを使用しないで外部データ ソースにアクセスする場合は、ADO を使用してください。 Connections コレクションには、Workspace オブジェクトの現在の Connection オブジェクトが含まれています (ODBCDirect ワークスペースのみ)。
