# <a name="caption-property-ado-md"></a>Caption プロパティ (ADO MD)


**適用されます**Access 2013、Office 2013。

[Level](level-object-ado-md.md) オブジェクト、または [Member](member-object-ado-md.md) オブジェクトを表示するときに使用するキャプションのテキストを示します。
