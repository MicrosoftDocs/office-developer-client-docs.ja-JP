# <a name="fieldsourcefield-property-dao"></a>フィールドの sourcefield プロパティ (DAO)


**適用先:** Access 2013、Office 2013

**Field** オブジェクトの元のデータ ソースであるフィールド名を示す値を取得します。値の取得のみ可能です。文字列型 ( **String**) の値を使用します。

## <a name="syntax"></a>構文

*式*。sourcefield プロパティ

*式***Field**オブジェクトを表す変数を取得します。
